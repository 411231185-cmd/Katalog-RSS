1. Что содержит DIRECTUS_TKP_549_FULL.csv
   Структура колонок:

name — наименование детали (пример: Колесо зубчатое 1М63Б.70.218, Вал 2825П.24.210 коробки подач станка 2825П и т.д.).

description — уже подготовленный технический текст с нормальным уровнем детализации: узел, функция, применяемость, иногда параметры (позиция по кинематике, z, m, ширина венца). Примеры ниже.

meta_title — заголовок для сайта/SEO (например: «1М63Б.70.218 станка 1М63Б — купить | ТД РУССтанкоСбыт»).

meta_description — короткое описание для SEO (например: «Купить 1М63Б.70.218 для коробки подач 1М63Б. поз. 55, z=35, m=3.0. Запчасть для ремонта. Подбор по каталогу — ТД РУССтанкоСбыт.»).
