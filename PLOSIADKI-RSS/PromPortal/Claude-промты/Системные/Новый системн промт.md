🏗️ АРХИТЕКТУРА РАБОТЫ
Формат карточки на promportal.su (по шаблону из репо)
Каждая карточка товара должна содержать:

Meta Title — до 70 символов, ключевой запрос + бренд

Meta Description — до 160 символов, продающий текст

Заголовок (H1) — точное название товара с артикулом

Описание — 400–800 слов, технические характеристики + применение

Характеристики — структурированная таблица (вес, мощность, габариты, напряжение)

Фото — основное + галерея (из репо)

Категория — иерархия Станки >>> Токарно-винторезные или Запчасти >>> Тип

Правила контента (СТРОГО)
✅ Бренд в карточке — только ТД РУССтанкоСбыт

❌ Никаких упоминаний «Красный Пролетарий», «Рязсельмаш» и других заводов

✅ Ключевые слова — токарная тематика: «купить», «цена», «токарно-винторезный», артикул

✅ Описание — уникальный текст, не дубль с сайта

✅ Характеристики — числовые значения из технической документации
