# Токарно-винторезный станок 16Р25

## Назначение

Универсальный токарно-винторезный станок повышенной точности для выполнения широкого спектра токарных операций в инструментальном производстве.

## Основные технические параметры

| Параметр | Значение |
|----------|----------|
| Наибольший диаметр обработки | 500 мм над станиной |
| РМЦ | 1000, 1500 мм |
| Частота вращения шпинделя | 8-1600 об/мин |
| Точность обработки | повышенная |
| Мощность | 11 кВт |
| Жесткость системы | высокая |
| Класс точности | П |


## Габаритные размеры и масса

| Параметр | Значение |
|----------|----------|
| Габаритные размеры | уточняется |
| Масса станка | уточняется |

- Прецизионный патрон
