# Токарно-винторезный станок 1М63Н (ДИП-300, 163, 1М63)

## Назначение

Токарно-винторезный станок 1М63Н РМЦ 750−1500−2000−3000−4000−5000−8000−10000 мм предназначен для выполнения разнообразных токарных работ, включая точение конусов и нарезание резьб: метрических, дюймовых, модульных, питчевых. Универсальный станок для обработки деталей среднего размера в условиях единичного, мелкосерийного и серийного производства.
