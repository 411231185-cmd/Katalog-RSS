# Токарно-винторезный станок 1Н65 (ДИП-500, 165, 1М65)

## Назначение

Токарно-винторезный станок 1Н65 РМЦ 1000−3000−5000−6000−8000−10000−12000 мм предназначен для выполнения разнообразных токарных работ, включая точение конусов и нарезание резьб: метрических, дюймовых, модульных, питчевых. Тяжелый токарно-винторезный станок для обработки крупногабаритных деталей большой массы в единичном и серийном производстве.
