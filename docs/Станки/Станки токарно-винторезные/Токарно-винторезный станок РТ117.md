# Токарно-винторезный станок РТ117

## Назначение

Универсальный токарно-винторезный станок с расширенными технологическими возможностями для обработки деталей в условиях серийного производства.

## Основные технические параметры

| Параметр | Значение |
|----------|----------|
| Наибольший диаметр устанавливаемой заготовки над станиной | 1200.0 мм |
| Наибольший диаметр обрабатываемой заготовки над станиной | 1140.0 мм |
| Наибольший диаметр обрабатываемой заготовки над суппортом | 900.0 мм |
| Наибольшая длина обрабатываемой заготовки (РМЦ) | 1000,  3000,  5000,  8000,  10000 мм |
| Наибольший вес устанавливаемой заготовки | 8000,  10000 кг |
| Размер конца шпинделя передней бабки по DIN | 2-15М |
| Количество ступеней частот вращения | 24.0 |
| Диаметр цилиндрического отверстия в шпинделе | 128.0 мм |
| Пределы частот вращения шпинделя | 5 – 500 об/мин |
| Пределы рабочих подач продольных | 0,20 – 3,05 мм/об |
| Пределы рабочих подач поперечных | 0,07 – 1,04 мм/об |
| Пределы рабочих подач резцовых салазок | 0,07 – 1,04 мм/об |
| Ускоренное перемещение суппорта продольное | 3.0 м/мин |
| Ускоренное перемещение суппорта поперечное | 1.0 м/мин |
| Величина шагов нарезания резьб метрических | 1 – 120 мм |
| Величина шагов нарезания резьб дюймовых | 28 – 1/4 ниток/дюйм |
| Величина шагов нарезания резьб модульных | 0,5 – 30 модуль |
| Мощность главного привода | 22.0 кВт |
