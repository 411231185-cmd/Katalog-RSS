# Токарно-винторезный станок 16К20

## Назначение

Универсальный токарно-винторезный станок для токарной обработки деталей из различных материалов в условиях единичного и серийного производства.

## Основные технические параметры

| Параметр | Значение |
|----------|----------|
| Наибольший диаметр обработки над станиной | 400 мм |
| Наибольший диаметр обработки над суппортом | 220 мм |
| РМЦ | 750, 1000, 1500 мм |
| Количество подач | 42 продольных и 42 поперечных |
| Пределы подач | 0.05-2.8 мм/об |
| Частота вращения шпинделя | 12.5-2000 об/мин (22 ступени) |
| Мощность | 10 кВт |
| Класс точности | Н |


## Габаритные размеры и масса

| Параметр | Значение |
|----------|----------|
| Габаритные размеры | уточняется |
| Масса станка | уточняется |


## Стандартная комплектация

- Патрон 3-кулачковый 250 мм
- Центры передней и задней бабки
- Резцедержатель 4-позиционный
- Набор сменных шестерен
- Защитные экраны
- Светильник
- Насос СОЖ
- Комплект ЗИП и документации

## Дополнительное оснащение (опции)

- Люнеты (подвижные и неподвижные)
- Дополнительные инструментальные блоки
- Специализированная оснастка
- Система автоматической уборки стружки

## Особенности конструкции

- Высокая универсальность
- Широкий диапазон обработки
- Надежная конструкция
- Простота управления
- Доступность запчастей
- Низкая стоимость эксплуатации

## Область применения

- Обточка цилиндрических и конических поверхностей
- Нарезание метрических и дюймовых резьб
- Сверление и растачивание отверстий
