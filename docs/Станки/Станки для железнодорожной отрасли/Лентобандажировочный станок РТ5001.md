# Станок специальный лентобандажировочный РТ5001

## Назначение

Предназначен для бандажировки якорей электрических машин стеклолентой на эпоксидной основе. Обеспечивает высокую производительность и качество бандажировки.

## Основные технические параметры

| Параметр | Значение |
|----------|----------|
| Максимальный диаметр якоря | до 1000 мм |
| Максимальная длина якоря | до 2000 мм |
| Габариты | 2500×1200×1500 мм |
| Масса | около 1500 кг |
| Скорость вращения шпинделя | регулируемая |
| Тип привода бандажировки | Электрический |
| Мощность привода | 3 кВт |


## Габаритные размеры и масса

| Параметр | Значение |
|----------|----------|
| Габаритные размеры | 2500×1200×1500 мм |
| Масса станка | около 1500 кг |


## Стандартная комплектация

- Шпиндельная бабка с регулируемым приводом
- Задняя бабка с пневмозажимом
- Устройство подачи ленты
- Система нагрева
- Система охлаждения
- Пульт управления
- Защитные ограждения

## Дополнительное оснащение (опции)

- Люнеты (подвижные и неподвижные)
- Дополнительные инструментальные блоки
- Специализированная оснастка
- Система автоматической уборки стружки

## Особенности конструкции

- Высокая производительность
- Равномерная намотка
- Автоматический контроль натяжения
- Простота управления
- Надежная конструкция

## Область применения

- Бандажировка якорей электродвигателей Ремонт электрических машин Производство электротехнического оборудования
