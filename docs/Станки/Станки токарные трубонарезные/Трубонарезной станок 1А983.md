## Основные технические параметры

| Параметр | Значение |
|----------|----------|
| Диаметр обрабатываемых труб | 33-140 мм |
| Длина нарезаемой резьбы | до 200 мм |
| Частота вращения шпинделя | 31.5-315 об/мин |
| Мощность | 4.5 кВт |
| Тип резьбы | коническая трубная (К1/16) |
| Количество рабочих позиций | 2 |


## Габаритные размеры и масса

| Параметр | Значение |
|----------|----------|
| Габаритные размеры | уточняется |
| Масса станка | уточняется |


## Стандартная комплектация

- Резьбонарезные головки с гребенками
- Передняя бабка с зажимным устройством
- Задняя бабка с центром
- Копировальный суппорт
- Набор гребенок для различных резьб
- Система СОЖ
- Измерительный инструмент
- Комплект ЗИП

## Дополнительное оснащение (опции)

- Люнеты (подвижные и неподвижные)
- Дополнительные инструментальные блоки
- Специализированная оснастка
- Система автоматической уборки стружки

## Особенности конструкции

- Специализация на трубных резьбах
- Высокая точность резьбы
- Соответствие стандартам API
- Производительность
- Надежность конструкции
- Простота обслуживания

## Область применения

- Нарезка трубной конической резьбы Обработка труб для нефтегазовой отрасли Производство обсадных труб
