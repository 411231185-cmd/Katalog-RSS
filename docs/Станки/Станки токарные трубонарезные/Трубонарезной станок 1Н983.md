# Специальный трубонарезной станок 1Н983

## Назначение

Модернизированный трубонарезной станок для нарезания конических и цилиндрических резьб на трубах различного диаметра.

## Основные технические параметры

| Параметр | Значение |
|----------|----------|
| Диаметр труб | 48-178 мм |
| Длина резьбы | до 250 мм |
| Частота вращения | 25-250 об/мин |
| Точность резьбы | соответствует API |
| Мощность привода | 7.5 кВт |
| Типы резьб | коническая и цилиндрическая трубная |
| Класс точности | повышенный |


## Габаритные размеры и масса

| Параметр | Значение |
|----------|----------|
| Габаритные размеры | уточняется |
| Масса станка | уточняется |
