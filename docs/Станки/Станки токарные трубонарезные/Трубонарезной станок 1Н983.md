Специальный трубонарезной станок 1Н983
Назначение и возможности
 Модернизированный специализированный трубонарезной станок 1Н983 предназначен для нарезания конических и цилиндрических резьб на трубах нефтяного, газового и водяного сортамента в условиях единичного и мелкосерийного производства.

Станок 1Н983 является усовершенствованной моделью станка 1А983 с улучшенными характеристиками и расширенным функционалом для предприятий нефтегазовой отрасли.
На станке можно производить:
Нарезку трубных резьб по международным стандартам API 5B и ГОСТ 633-80 для обеспечения герметичности соединений
Обработку насосно-компрессорных труб (НКТ, НКМ), обсадных труб (ОТТМ, ОТТГ, Батресс) по ГОСТ 832-80
Производство труб для бурения — нарезка замковой резьбы на бурильных трубах, переводниках и штангах
Обточку концов труб, нарезание метрических, дюймовых, модульных и питчевых резьб, включая конические
Токарную обработку деталей трубных соединений — муфт, замков и переводников для нефтедобывающей промышленности
Станок оснащен конусной линейкой для обработки конических поверхностей и нарезания конических резьб. Обрабатываемая труба пропускается через полый шпиндель диаметром 300–320 мм и зажимается двумя механизированными патронами. Другой конец трубы поддерживается люнетом для обеспечения стабильности при обработке длинных заготовок. Шпиндель имеет прямое и обратное вращение для оптимизации процесса резьбонарезания.
