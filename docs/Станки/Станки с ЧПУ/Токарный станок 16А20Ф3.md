# Станок токарный патронно-центровой с числовым программным управлением 16А20Ф3

## Назначение

Универсальный токарный станок с ЧПУ среднего размера для автоматической обработки деталей в серийном производстве.

## Основные технические параметры

| Параметр | Значение |
|----------|----------|
| Наибольший диаметр устанавливаемой заготовки над станиной | 500.0 мм |
| Наибольший диаметр обрабатываемой заготовки над станиной | 320.0 мм |
| Наибольший диаметр обрабатываемой заготовки над суппортом | 200.0 мм |
| Наибольшая длина обрабатываемой заготовки (РМЦ) | 1000,  1500 мм |
| Диаметр цилиндрического отверстия в шпинделе | 55.0 мм |
| Пределы частот вращения шпинделя | 20 – 2500 об/мин |
| Количество управляемых координат | 2.0 |
| Точность позиционирования | 0.01 мм |
| Количество позиций инструментальной головки | 6, 8 |
| Мощность главного привода | 11.0 кВт |


## Габаритные размеры и масса

| Параметр | Значение |
|----------|----------|
| Длина | 3700,  4215 мм |
| Ширина | 2260.0 мм |

## Стандартная комплектация

- Система ЧПУ с графическим дисплеем
- Револьверная головка
- Патрон с автозажимом
- Задняя бабка программируемая
- СОЖ с баком
- Стружкоудаление
- Защита
- Пульт

- Дополнительные инструментальные блоки
- Специализированная оснастка
- Система автоматической уборки стружки
