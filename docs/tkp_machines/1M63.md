# Токарно-винторезный станок 1М63Н (ДИП-300, 163, 1М63)

## Общая информация

**Модель:** 1M63  
**Категория:** Универсальные токарные станки  
**Производитель:** ТД РУССтанкоСбыт

## Назначение

Универсальный токарно-винторезный станок для обработки деталей среднего размера в условиях единичного, мелкосерийного и серийного производства.

## Область применения

- Обточка цилиндрических и конических поверхностей
- Нарезание метрических, дюймовых, модульных и питчевых резьб
- Сверление, зенкерование, развертывание отверстий
- Подрезка торцов, отрезка
- Обработка с ЧПУ (модификация ДИП-300)

## Технические характеристики

| Параметр | Значение |

| Наибольший диаметр обработки над станиной | 630 мм |
| Наибольший диаметр обработки над суппортом | 355 мм |
