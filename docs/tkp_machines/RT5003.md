# Станок специальный лентобандажировочный РТ5003

## Общая информация

**Модель:** RT5003  
**Категория:** Специальное оборудование  
**Производитель:** ТД РУССтанкоСбыт

## Назначение

Универсальный лентобандажировочный станок для бандажировки якорей различных размеров. Модернизированная версия РТ5001 с расширенными возможностями.

## Область применения

- Бандажировка якорей крупногабаритных электродвигателей
- Ремонтное производство
- Серийное изготовление электрических машин
