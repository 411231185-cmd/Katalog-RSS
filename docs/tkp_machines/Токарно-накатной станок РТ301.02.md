# Специальный токарно-накатной станок РТ301.02

## ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ

### Назначение и возможности

Специализированный станок для накатки резьб и рифлений с автоматическим циклом работы. Повышенная производительность.

Накатка метрических резьб Накатка специальных профилей Серийное производство
