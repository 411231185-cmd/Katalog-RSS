## Для Tilda

Эти ТКП готовы для использования на посадочных страницах Tilda:
1. Скопируйте содержимое нужного MD файла
2. Конвертируйте Markdown в HTML (если необходимо)
3. Вставьте в блоки Tilda
4. Настройте дизайн согласно брендбуку
