# Станок токарный трубонарезной с числовым программным управлением РТ779Ф3

## ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ

### Назначение и возможности

Специализированный трубонарезной станок с ЧПУ для автоматической нарезки резьбы на трубах нефтяного сортамента и обработки труб.

Нарезка резьбы на трубах Обработка труб нефтяного сортамента Автоматическое производство труб

### Размеры и масса обработки

**Длина нарезаемой резьбы:** до 300 мм
