## Назначение

Специализированный трубонарезной станок с ЧПУ для автоматической нарезки резьбы на трубах нефтяного сортамента и обработки труб.

- Нарезка резьбы на трубах
- Обработка труб нефтяного сортамента
- Автоматическое производство труб
- Серийная нарезка трубных резьб
