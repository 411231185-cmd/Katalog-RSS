# Станок токарно-давильный с числовым программным управлением РТ305М

## Общая информация

**Модель:** RT305M  
**Категория:** Специальные станки с ЧПУ  
**Производитель:** ТД РУССтанкоСбыт

## Назначение

Специализированный станок с ЧПУ для давильных работ по изготовлению деталей из листового металла методом ротационной вытяжки.

## Область применения

- Ротационная вытяжка деталей
- Изготовление корпусов и обечаек
- Производство посуды и емкостей
- Обработка листовых заготовок
- Давильные работы по программе
