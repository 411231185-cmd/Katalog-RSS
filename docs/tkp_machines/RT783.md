# Специальный трубонарезной станок РТ783

## Общая информация

**Модель:** RT783  
**Категория:** Специальные трубонарезные станки  
**Производитель:** ТД РУССтанкоСбыт

## Назначение

Современный трубонарезной станок с гидравлическим приводом для высокопроизводительной нарезки резьб на трубах большого диаметра.

## Область применения

- Нарезка резьб на трубах большого диаметра
- Производство обсадных колонн
- Обработка труб для магистральных трубопроводов
- Нефтегазовое машиностроение

## Технические характеристики

| Параметр | Значение |
|----------|----------|
| Диаметр обрабатываемых труб | 60-245 мм |
| Максимальная длина резьбы | до 300 мм |
| Тип привода | гидравлический |
| Типы нарезаемых резьб | трубные конические и цилиндрические по API |
| Частота вращения | регулируемая 20-200 об/мин |
| Мощность | 15 кВт |
| Усилие зажима | до 100 кН |
| Производительность | до 150 труб/смену |
