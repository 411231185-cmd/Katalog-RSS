# Специальный трубонарезной станок 1А983

## Общая информация

**Модель:** 1A983  
**Категория:** Специальные трубонарезные станки  
**Производитель:** ТД РУССтанкоСбыт

## Назначение

Специализированный станок для нарезания конической трубной резьбы на трубах нефтяного сортамента. Обеспечивает высокую точность резьбы.

## Область применения

- Нарезка трубной конической резьбы
- Обработка труб для нефтегазовой отрасли
- Производство обсадных труб
- Ремонт трубопроводов

## Технические характеристики

| Параметр | Значение |
|----------|----------|
| Диаметр обрабатываемых труб | 33-140 мм |
| Длина нарезаемой резьбы | до 200 мм |
| Тип резьбы | коническая трубная (К1/16) |
| Частота вращения шпинделя | 31.5-315 об/мин |
| Мощность | 4.5 кВт |
| Количество рабочих позиций | 2 |
| Производительность | до 120 труб/смену |
