# Резцовые блоки

## ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ

### Назначение и возможности

Резцовые блоки для токарных станков с ЧПУ - прецизионные инструментальные блоки для крепления режущего инструмента в револьверных головках. Обеспечивают быструю смену инструмента, высокую точность позиционирования и жесткость закрепления резцов. Совместимы с различными системами инструментальной оснастки.

### Область применения

- Токарная обработка на станках с ЧПУ
- Многопозиционное точение
- Автоматическая смена инструмента
- Высокопроизводительная обработка
- Серийное производство деталей

## ОСНОВНЫЕ ТЕХНИЧЕСКИЕ ПАРАМЕТРЫ

**Типы блоков:** Радиальные, осевые, угловые

**Размер резца:** 16×16, 20×20, 25×25, 32×32 мм

**Материал корпуса:** Сталь 40Х, закалка HRC 50-55

**Точность позиционирования:** ±0.01 мм

**Повторяемость установки:** ±0.005 мм

**Совместимость:** Револьверные головки VDI, BMT, SAUTER

**Типы крепления:** VDI 20, VDI 30, VDI 40, BMT 45, BMT 55

**Максимальная скорость:** до 6000 об/мин
