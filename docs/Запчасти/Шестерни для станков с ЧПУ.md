# Шестерни для станков с ЧПУ

## ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ

### Назначение и возможности

Комплектующие зубчатые колеса и шестерни для станков с числовым программным управлением. Высокоточные детали для коробок скоростей, шпиндельных бабок, приводов подач современных токарных и фрезерных станков с ЧПУ. Изготовление с применением прецизионного оборудования и контрольно-измерительных приборов.

### Область применения

- Ремонт станков с ЧПУ
- Модернизация привода главного движения
- Замена изношенных зубчатых передач
- Восстановление коробок скоростей
- Создание запаса критичных деталей
