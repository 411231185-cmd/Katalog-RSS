## Использование для Tilda

Эти ТКП готовы для использования на посадочных страницах Tilda:

1. Скопируйте содержимое нужного MD файла
2. Вставьте секции в соответствующие блоки Tilda
3. Настройте дизайн согласно брендбуку
4. Опубликуйте страницу
