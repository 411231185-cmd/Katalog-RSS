# Система ЧПУ

Система числового программного управления (ЧПУ) управляет всеми осями и исполнительными механизмами станка. Включает стойку, панель оператора, блок питания и интерфейсные модули.

## Состав узла (типовой)

- Стойка ЧПУ (NC210/Siemens/Fagor — вариант по комплектации)
- Панель оператора (HMI/экран/клавиатура)
- Блок питания
- PLC/модуль I/O
- Интерфейсные модули (если применимо)
- Батарея/память (если применимо)
