# Ось Z (ШВП)

Ось Z с шарико-винтовой парой (ШВП) обеспечивает продольное перемещение суппорта с высокой точностью и минимальным люфтом. Используется в ЧПУ-исполнениях для передачи движения от серводвигателя к рабочему органу.

## Состав узла (типовой)

- ШВП продольная (винт)
- Гайка ШВП (или каретка гайки)
- Опорные подшипники ШВП (комплект)
- Корпус/крышка опоры (если применимо)
- Муфта “серво—ШВП”
- Защита ШВП (телескоп/гофра/кожух)
