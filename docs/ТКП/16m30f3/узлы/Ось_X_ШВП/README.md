# Ось X (ШВП)

Ось X с шарико-винтовой парой (ШВП) обеспечивает поперечное перемещение суппорта или резцедержателя. Гарантирует высокую точность позиционирования и повторяемость.

## Состав узла (типовой)

- ШВП поперечная (винт)
- Гайка ШВП
- Опорные подшипники ШВП
- Муфта “серво—ШВП”
- Защита ШВП
