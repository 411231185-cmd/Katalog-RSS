# Серво Z

Серводвигатель оси Z обеспечивает привод продольного перемещения. Включает двигатель, драйвер, энкодер и кабельную обвязку.

## Состав узла (типовой)

- Серводвигатель оси Z
- Сервоусилитель/драйвер
- Энкодер (если отдельный)
- Тормоз (если применимо)
- Кабели силовые/энкодерные (комплект)
