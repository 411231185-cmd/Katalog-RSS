# Суппорт поперечный 1Н65

Суппорт поперечный обеспечивает поперечное перемещение резцедержателя, точность подачи и фиксацию инструмента. Включает винты, гайки и элементы регулировки.

## Состав узла (10–15)

- Винт поперечной подачи суппорта
- Гайка винта поперечной подачи
- Клин поперечных салазок
- Прижимные планки/сухари направляющих
- Маховик поперечной подачи
- Лимб/шкала
- Подшипники/втулки опоры винта
