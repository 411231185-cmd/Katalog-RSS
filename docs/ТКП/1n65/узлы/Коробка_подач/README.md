# Коробка подач 1Н65

Коробка подач предназначена для передачи и изменения скорости подачи суппорта и каретки. Содержит валы, шестерни и механизмы переключения.

## Состав узла (10–15)

- Входной вал (от гитары)
- Промежуточные валы
- Выходной вал (на ходовой вал/фартук)
- Блок‑шестерни коробки подач
- Шестерни постоянного зацепления
- Вилки переключения
- Штоки/валы переключения
- Фиксаторы положений (комплект)
- Подшипники валов
- Втулки (где применены)
- Плунжерный насос смазки коробки подач
- Фильтр смазки (сервис)
- Маслоуказатель/уровнемер (сервис)
- Комплект настройки 2‑го ряда подач (a=42; b=c=126 для 1Н65)
- Сменные колёса “точных резьб”
