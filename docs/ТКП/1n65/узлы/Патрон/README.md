# Патрон 1Н65

Патрон предназначен для закрепления заготовки на шпинделе, обеспечивает быструю смену и надёжную фиксацию. Включает кулачки, крепёж и переходные элементы.

## Состав узла (10–15)

- Кулачки (прямые/обратные — комплекты)
- Винты/крепеж кулачков
- Пары “улитка/рейка” или винты кулачков (ЗИП‑группа)
- Ключ патронный
- Переходной фланец/планшайба
