# Опоры ходового винта (кронштейн правой части)

## ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ

| Параметр        | Значение                                                                 |
| --------------- | ------------------------------------------------------------------------ |
| Расположение    | Правая часть станины, передняя стенка (в одном кронштейне с опорой вала) |
| Конструкция     | Радиально-упорный подшипник + втулка                                     |
| Диаметр посадки | под винт Ø70 мм (шейка)                                                  |
| Тип подшипника  | Радиально-упорный шариковый (класс точности 5)                           |

## Назначение и возможности

Опора правого конца ходового винта воспринимает радиальные и осевые нагрузки, обеспечивая точное вращение винта без осевого люфта. Осевая фиксация критически важна для точности нарезаемой резьбы.

## Область применения

- Базирование правого конца ходового винта
- Восприятие осевых усилий при нарезании резьбы
- Станки: 1Н65, 1М65 всех модификаций

## ОСНОВНЫЕ ТЕХНИЧЕСКИЕ ПАРАМЕТРЫ

- Посадочный диаметр: под шейку винта (расточка H7)
- Осевой люфт: регулируется гайками, допуск не более 0,01 мм
- Подшипник: радиально-упорный, класс точности 5 по ГОСТ 520

## Материалы и покрытия

- **Кронштейн:** чугун СЧ20
- **Подшипник:** ШХ15 (стандартный)
- **Регулировочные гайки:** сталь 45, оксидирование

## КОМПЛЕКТАЦИЯ

- 📋 Кронштейн с установленными подшипниками
- 🔧 Регулировочные гайки и контргайки
- 🔧 Крепёж
- 📐 Паспорт

## ПРЕИМУЩЕСТВА

- ✅ Прецизионные подшипники — минимальный осевой люфт
- ✅ Регулировка без разборки станины
- ✅ Совместимость с оригиналом
