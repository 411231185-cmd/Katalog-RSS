# Узел крепления-прихватов (планки-болты крепления узлов на станине)

## ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ

| Параметр        | Значение                                                   |
| --------------- | ---------------------------------------------------------- |
| Тип             | Комплект крепёжных элементов для фиксации узлов на станине |
| Состав          | Планки прижимные, болты, гайки, шайбы                      |
| Размер болтов   | М16, М20, М24 (основные)                                   |
| Класс прочности | 8.8–10.9                                                   |
| Покрытие        | Оксидирование или цинкование                               |

## Назначение и возможности

Комплект крепёжных элементов (прихваты) используется для фиксации на станине: задней бабки, люнетов неподвижных, конусной линейки и других устройств. Прижимные планки и болты обеспечивают надёжное крепление узлов в Т-образных пазах станины.

## Область применения

- Крепление задней бабки на станине
- Установка и фиксация неподвижного люнета
- Крепление конусной линейки
- Фиксация ограждений и кожухов
- Станки: 1Н65, 1М65, 1А64, ДИП-500

## ОСНОВНЫЕ ТЕХНИЧЕСКИЕ ПАРАМЕТРЫ

- Болты: М16×80, М20×100, М24×120 (типовые)
- Планки: длина 80–150 мм, толщина 12–16 мм
- Класс прочности болтов: 8.8 (ГОСТ 7798)
- Гайки: класс 8 (ГОСТ 5915)
- Шайбы: пружинные (Гровера) ГОСТ 6402

## Материалы и покрытия

- **Болты и гайки:** сталь 35, 40Х, класс прочности 8.8–10.9
- **Планки:** сталь 45, нормализация
- **Покрытие:** оксидирование (чернение) или цинкование

## КОМПЛЕКТАЦИЯ

Типовой комплект:

- 📋 Планки прижимные — 4–6 шт.
- 🔧 Болты с Т-образной головкой — 4–6 шт.
- 🔧 Гайки — 4–6 шт.
- 🔧 Шайбы пружинные — 4–6 шт.
- 📐 Паспорт комплекта

- ✅ Высокий класс прочности — надёжная фиксация тяжёлых узлов
- ✅ Антикоррозийное покрытие
