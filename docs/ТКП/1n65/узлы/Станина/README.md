# Станина 1Н65

Станина служит основой станка, обеспечивает жёсткость конструкции и точность перемещений. На ней расположены направляющие для перемещения суппорта, каретки и других узлов.

## Состав узла (10–15)

- Направляющие (износ/восстановление: шлифовка/накладки)
- Рейка продольного перемещения
- Ходовой винт
- Ходовой вал
- Опоры/подшипники ходового винта
- Опоры/подшипники ходового вала
- Подвеска(и) ходового винта/вала
- Фитильная смазка правой опоры ходового винта/вала (комплект)
- Очистители направляющих (щетки/скребки)
- Слив/сетка корыта
