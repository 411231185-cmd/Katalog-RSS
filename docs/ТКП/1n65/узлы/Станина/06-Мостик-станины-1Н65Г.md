# Мостик станины 1Н65Г

**Описание:**
Съёмная вставка для выемки в станине исполнения «Г».

**Технические характеристики:**

- Обозначение: Мостик станины 1Н65Г
- Назначение: Заполнение выемки в станине при необходимости
- Тип установки: Съёмный, крепится болтами
- Длина выемки: 390 мм
- Диаметр заготовки над выемкой: до 1400 мм
- Масса мостика: ~80–120 кг

**Назначение:**
Восстановление непрерывности направляющих, обработка крупногабаритных деталей.

**Преимущества:**

- Совпадение направляющих: допуск ±0,01 мм по высоте
- Крепление: 4–6 болтов + штифты для точного базирования

**Материал:**
Чугун СЧ20 (аналогично станине)
