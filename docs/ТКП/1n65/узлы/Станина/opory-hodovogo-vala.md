# Опоры ходового вала (кронштейн правой части)

**Описание:**
Кронштейн с запрессованными подшипниковыми втулками для фиксации заднего конца ходового вала.

**Технические характеристики:**

- Расположение: Правая (задняя) часть станины, передняя стенка
- Конструкция: Кронштейн с подшипниковыми втулками
- Тип подшипника: Скольжения (бронзовая втулка) или качения
- Диаметр посадки: под вал Ø55 мм
- Крепление: Болтами через фланец

**Назначение:**
Фиксация заднего конца вала, обеспечение соосности с механизмом фартука и коробкой подач.

**Преимущества:**

- Бронзовая втулка — минимальное трение
- Точная расточка — соосность с левой опорой
- Простая замена втулки при износе

**Комплектация:**

- Кронштейн в сборе с подшипниковой втулкой
- Крепёжные болты
- Паспорт
