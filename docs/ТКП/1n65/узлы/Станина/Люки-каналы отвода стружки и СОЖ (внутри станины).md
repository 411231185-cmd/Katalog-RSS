# Люки-каналы отвода стружки и СОЖ (внутри станины)

### ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ

| Параметр           | Значение                                  |
| ------------------ | ----------------------------------------- |
| Тип                | Наклонные люки внутри корпуса станины     |
| Количество         | 4–6 шт. (зависит от РМЦ)                  |
| Угол наклона       | ~30–45°                                   |
| Направление отвода | В сторону, противоположную рабочему месту |
| Материал           | Чугун СЧ20 (литьё заодно со станиной)     |

[^2]

### Назначение и возможности

Наклонные люки внутри станины обеспечивают гравитационный отвод стружки и отработанной СОЖ из зоны обработки в корыто-ванну. Наклонная конструкция предотвращает накопление стружки внутри станины, что критически важно для тяжёлых режимов резания с большим объёмом стружкообразования.[^2]

### Область применения

- Отвод витой и сыпучей стружки из зоны резания
- Отвод СОЖ в ёмкость рециркуляции
- Защита внутренних механизмов станины от попадания стружки
- Все модификации 1Н65[^2]

### ОСНОВНЫЕ ТЕХНИЧЕСКИЕ ПАРАМЕТРЫ

- Угол наклона: 30–45° (обеспечивает самосход стружки)
- Ширина прохода: достаточная для витой стружки Ø до 50 мм
- Поверхность: гладкая литая, без задиров

### Материалы и покрытия

- **Материал:** чугун СЧ20 (ГОСТ 1412-85), литьё заодно с корпусом станины
- **Покрытие:** антикоррозийная грунтовка

### КОМПЛЕКТАЦИЯ

- 📋 Люки — элемент станины (не поставляются отдельно)
- 🔧 При ремонте: восстановление поверхностей, зачистка наплывов
- 📐 Дефектовка при капремонте станины

### ПРЕИМУЩЕСТВА

- ✅ Монолитная конструкция — надёжность
- ✅ Оптимальный угол наклона — самосход стружки
- ✅ Не требуют обслуживания при нормальной эксплуатации

### ЗАКАЗ И ПОСТАВКА

- 🚚 Услуга ремонта/зачистки люков при капремонте станины
- ⏱️ В составе работ по восстановлению станины
