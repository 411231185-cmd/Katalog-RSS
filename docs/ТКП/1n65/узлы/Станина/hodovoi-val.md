# Ходовой вал

**Описание:**
Ходовой вал — элемент передачи вращательного движения. Служит для синхронизации работы узлов.

**Технические характеристики:**

- Материал: Сталь 40Х
- Длина: 1150 мм
- Диаметр: 35 мм

**Особенности:**

- Повышенная прочность
- Минимальный люфт

**Применение:**
Обеспечивает передачу вращения между механизмами.
