# Мостик (для выемки в станине, исполнение «Г»)

## ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ

| Параметр                          | Значение                                      |
| --------------------------------- | --------------------------------------------- |
| Обозначение                       | Мостик станины 1Н65Г                          |
| Назначение                        | Заполнение выемки в станине при необходимости |
| Тип установки                     | Съёмный, крепится болтами                     |
| Длина выемки (от зеркала патрона) | 390 мм                                        |
| Диаметр заготовки над выемкой     | до 1400 мм                                    |
| Масса мостика                     | ~80–120 кг                                    |

## Назначение и возможности

Мостик — съёмная вставка, устанавливаемая в выемку станины станков исполнения «Г» (1Н65Г, 1Н65ГФ1). Выемка позволяет обрабатывать заготовки увеличенного диаметра (до 1400 мм) непосредственно у патрона. Когда обработка крупногабаритных деталей не требуется, мостик устанавливается на место, восстанавливая непрерывность направляющих для нормального перемещения каретки.

## Область применения

- Станки исполнения «Г»: 1Н65Г, 1Н65ГФ1
- Обработка деталей типа «диск» / «фланец» Ø до 1400 мм при снятом мостике
- Восстановление функции направляющих при установке мостика

## ОСНОВНЫЕ ТЕХНИЧЕСКИЕ ПАРАМЕТРЫ

- Длина: ~390 мм (равна длине выемки)
- Совпадение направляющих: допуск ±0,01 мм по высоте
- Крепление: 4–6 болтов + штифты для точного базирования

## Материалы и покрытия

- **Материал:** чугун СЧ20 (аналогично станине)
- **Направляющие поверхности:** закалка ТВЧ HRC 42–50, шлифовка
- **Общее покрытие:** грунтовка + эмаль
