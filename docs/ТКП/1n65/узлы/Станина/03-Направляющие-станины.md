# Направляющие станины

**Описание:**
Направляющие станины обеспечивают точность перемещения подвижных узлов.

**Технические характеристики:**

- Материал: Сталь 45
- Длина: 1200 мм
- Ширина: 60 мм
- Толщина: 20 мм

**Особенности:**

- Прецизионная обработка
- Износостойкое покрытие

**Применение:**
Используются для направления движения каретки и других подвижных частей.
