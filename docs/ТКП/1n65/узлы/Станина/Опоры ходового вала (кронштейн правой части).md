# Опоры ходового вала (кронштейн правой части)

## ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ

| Параметр            | Значение                                            |
| ------------------- | --------------------------------------------------- |
| Расположение        | Правая (задняя) часть станины, передняя стенка      |
| Конструкция         | Кронштейн с запрессованными подшипниковыми втулками |
| Тип подшипника      | Скольжения (бронзовая втулка) или качения           |
| Диаметр посадки     | под вал Ø55 мм                                      |
| Крепление к станине | Болтами через фланец                                |

## Назначение и возможности

Кронштейн правой опоры ходового вала фиксирует задний конец вала, воспринимая радиальные нагрузки и предотвращая провисание. Расположен на передней стенке правой части станины. Обеспечивает соосность вала с механизмом фартука и коробкой подач.

## Область применения

- Опора правого конца ходового вала на станках с РМЦ 1000–5000 мм
- Станки: 1Н65, 1М65, 1А64, ДИП-500

## ОСНОВНЫЕ ТЕХНИЧЕСКИЕ ПАРАМЕТРЫ

- Посадочный диаметр: Ø55 H7
- Допуск соосности с левой опорой: 0,03 мм
- Подшипник скольжения: бронза БрОЦС5-5-5

## Материалы и покрытия

- **Кронштейн:** чугун СЧ20 или сталь 35Л
- **Втулка подшипника:** бронза БрОЦС5-5-5 (ГОСТ 613-79)
- **Покрытие кронштейна:** грунтовка + эмаль

## КОМПЛЕКТАЦИЯ

- 🔧 Крепёжные болты
