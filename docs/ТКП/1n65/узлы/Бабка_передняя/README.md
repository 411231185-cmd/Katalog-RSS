# Бабка передняя 1Н65

Передняя бабка содержит коробку скоростей, обеспечивает вращение шпинделя и передачу движения на подачу. В ней расположены основные механизмы переключения скоростей и передачи крутящего момента.

## Состав узла (10–15)

- Ведущий вал коробки скоростей
- Промежуточные валы
- Выходной вал (на привод подач/гитару)
- Блок‑шестерни (переключаемые)
- Шестерни постоянного зацепления
- Вилки переключения
- Тяги/рычаги переключения
- Узел “увеличение шага ×8” (комплект)
- Реверсивный блок
- Электромагнитная тормозная муфта шпинделя (в сборе/катушка/якорь)
- Подшипники валов коробки скоростей
- Маслонасос смазки передней бабки
- Фильтр/клапан системы смазки
