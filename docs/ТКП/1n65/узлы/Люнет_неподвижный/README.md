# Люнет неподвижный 1Н65

Люнет неподвижный фиксируется на станине и поддерживает заготовку в процессе обработки. Включает сухари, ролики и винты регулировки.

## Состав узла (10–15)

- Сухари (комплект)
- Ролики (если применимо)
- Подшипники роликов
- Винты регулировки/прижима
- Гайки/контргайки регулировки
