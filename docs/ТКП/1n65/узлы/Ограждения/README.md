# Ограждения 1Н65

Ограждения обеспечивают безопасность работы, защищают оператора от движущихся частей и стружки. Включают кожухи, щитки и элементы крепления.

## Состав узла (10–15)

- Ограждение патрона
- Щиток зоны резания
- Кожух ременной передачи/шкивов (если применимо)
- Кожух сменных колёс (гитары)
- Блокировки/замки ограждений (если применимо)
- Крепеж/петли/фиксаторы щитков (комплект)
