# Шпиндельный узел 1Н65

Шпиндельный узел обеспечивает вращение заготовки, передачу крутящего момента и точность обработки. Включает шпиндель, подшипники и элементы крепления патрона.

## Состав узла (10–15)

- Шпиндель
- Подшипники шпинделя (передняя опора)
- Подшипники шпинделя (задняя опора)
- Подшипники шпинделя (средняя опора — если применимо)
- Регулировочные гайки/контргайки подшипников
- Дистанционные/упорные кольца
- Уплотнения шпинделя (манжеты/лабиринт)
- Шестерня/блок на шпинделе
- Присоединительные элементы под патрон (фланец/крепеж — если применимо)
