# Верхний суппорт 1Н65

Верхний суппорт предназначен для установки и точной настройки резцедержателя, обеспечивает поворот и перемещение инструмента. Включает винты, клинья и лимбы.

## Состав узла (10–15)

- Винт верхнего суппорта
- Гайка винта верхнего суппорта
- Клин верхнего суппорта
- Поворотная плита/основание (если применимо)
- Рукоятка/маховик верхнего суппорта
- Лимб/шкала
- Резцедержатель (4‑позиционный)
