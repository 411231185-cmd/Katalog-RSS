# Сменные колёса (гитара) 1Н65

Гитара сменных колёс обеспечивает передачу движения от коробки скоростей к коробке подач и настройку резьбовых подач. Включает комплект сменных шестерён и крепёж.

## Состав узла (10–15)

- Комплект сменных шестерён (гитара)
- Колесо сменное Z=44, M=2,5 (1Н65.08.154)
- Колесо сменное Z=58, M=2,5 (1Н65.08.155)
- Колесо сменное Z=44, M=2,5 (1Н65.08.156)
- Колесо сменное Z=88, M=2,5 (1Н65.08.157)
- Колесо сменное Z=88, M=2,5 (1Н65.08.158)
- Палец/ось гитары
- Втулки дистанционные под колёса
- Шпонки/штифты фиксации
- Гайки/шайбы крепления (комплект)
- Кронштейн гитары (если применимо)
