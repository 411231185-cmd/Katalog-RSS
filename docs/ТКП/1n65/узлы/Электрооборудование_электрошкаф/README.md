# Электрооборудование электрошкаф 1Н65

Электрошкаф содержит элементы управления и защиты электропитания станка. Включает автоматы, контакторы, реле, трансформаторы и кабельную разводку.

## Состав узла (10–15)

- Вводной автомат/рубильник
- Контакторы/пускатели главного привода
- Тепловые реле/защита
- Трансформатор управления/освещения 24В
- Кнопки “Пуск/Стоп/Аварийный стоп”
- Переключатели (в т.ч. СОЖ)
- Реле/промежуточные реле
- Предохранители/клеммники
- Кабели/разъёмы/кабель‑каналы
- Концевики/блокировки ограждений (если применимо)
