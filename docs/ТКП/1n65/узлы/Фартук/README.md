# Фартук 1Н65

Фартук обеспечивает автоматическую и ручную подачу суппорта, содержит механизмы муфт, валов и насосов смазки. Важен для точности и безопасности работы.

## Состав узла (10–15)

- Электромагнитные муфты фартука (4 муфты для 1Н65)
- Полугайка/гайка ходового винта (разрезная)
- Обгонная муфта
- Валы фартука
- Шестерни цепей продольной/поперечной подачи
- Механизм предохранительной муфты
- Электродвигатель быстрых ходов (если применимо)
- Плунжерный насос смазки фартука
- Эксцентрик привода насоса
- Подшипники/втулки валов фартука
