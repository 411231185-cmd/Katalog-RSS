# Каретка 1Н65

Каретка перемещается по направляющим станины, обеспечивает поперечную подачу и поддерживает суппорт. Включает винты, гайки и механизмы регулировки.

## Состав узла (10–15)

- Винт поперечной подачи каретки
- Гайки винта поперечной подачи (комплект)
- Шестерня ручного привода от рейки
- Клинья регулировки направляющих каретки
- Прижимные планки/сухари
- Маховик ручного перемещения каретки
- Муфта УЦИ (сильфонная) — если применимо
- Датчик/линейка УЦИ — если применимо
