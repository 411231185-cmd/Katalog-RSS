# Бабка задняя 1Н65

Задняя бабка поддерживает длинные заготовки, обеспечивает точность центрирования и перемещения пиноли. Включает пиноль, винтовую пару и механизмы фиксации.

## Состав узла (10–15)

- Пиноль
- Шпиндель задней бабки (внутренний)
- Винтовая пара перемещения пиноли (винт+гайка)
- Маховик/штурвал
- Муфта зубчатая
- Вилка включения
- Червяк редуктора
- Шестерни редуктора
- Фиксатор/эксцентрик зажима
- Центр
