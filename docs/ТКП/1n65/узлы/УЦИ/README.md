# УЦИ 1Н65

Устройство цифровой индикации (УЦИ) обеспечивает точный контроль перемещений, отображает координаты и параметры обработки. Включает блок индикации, датчики и кабели.

## Состав узла (10–15)

- Блок индикации
- Линейка/датчик (ось X)
- Кабель датчика
- Кронштейны крепления
- Муфта соединительная (если применимо)
- Блок питания/электроника (ремонтная группа)
