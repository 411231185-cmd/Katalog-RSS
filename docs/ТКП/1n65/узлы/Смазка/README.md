# Смазка 1Н65

Система смазки обеспечивает подачу масла к основным узлам станка, снижает трение и износ. Включает насосы, фильтры, маслоуказатели и трубопроводы.

## Состав узла (10–15)

- Насос смазки передней бабки (циркуляционный)
- Фильтры смазки
- Предохранительный клапан
- Маслоуказатели потока (МУ1…МУ3)
- Указатели уровня масла (УУЖ)
- Плунжерный насос коробки подач (Н1)
- Плунжерный насос фартука (Н2)
- Ручной плунжерный насос смазки направляющих
- Маслёнки (комплект)
- Фитили/фитильные узлы
- Маслопроводы/трубки/штуцеры (ремкомплект)
