# СОЖ 1Н65

Система охлаждения и смазки (СОЖ) обеспечивает подачу эмульсии в зону резания, снижает износ инструмента и удаляет стружку. Включает насос, резервуар и трубопроводы.

## Состав узла (10–15)

- Электронасос СОЖ
- Резервуар/бак эмульсии
- Корыто/ванна станины
- Сетки/фильтрующие решетки
- Трубопровод подачи
- Гибкий шланг к резцу
- Сопло/насадка подачи
- Кран/регулятор расхода
- Электрика насоса (как узел электрики)
