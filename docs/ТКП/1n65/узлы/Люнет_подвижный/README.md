# Люнет подвижный 1Н65

Люнет подвижный поддерживает длинные детали при обработке, предотвращает прогиб и вибрации. Включает сухари, ролики и крепёж.

## Состав узла (10–15)

- Сухари верхние
- Сухари нижние сменные
- Винты прижима сухарей
- Гайки винтов
- Ролики (если применимо)
- Подшипники роликов (если применимо)
