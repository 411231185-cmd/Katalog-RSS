## Обзор

## Формат ТКП

Каждый ТКП включает следующие разделы:

- **ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ** - Назначение и возможности
- **Область применения** - Конкретные области использования
- **ОСНОВНЫЕ ТЕХНИЧЕСКИЕ ПАРАМЕТРЫ** - Подробные спецификации
- **КОМПЛЕКТАЦИЯ** - Состав поставки с emoji иконками
- **ПРЕИМУЩЕСТВА** - Ключевые преимущества с галочками ✅
- **ЗАКАЗ И ПОСТАВКА** - Информация о заказе и доставке
- **ГАРАНТИИ И КАЧЕСТВО** - Гарантийные обязательства
- **КОНТАКТНАЯ ИНФОРМАЦИЯ** - Полные контактные данные

## Использование

Эти ТКП готовы для:

1. **Прямого копирования** в блоки Tilda
2. **Создания посадочных страниц** для рекламных кампаний
3. **Формирования коммерческих предложений** для клиентов
4. **Базы знаний** ТД РУССтанкоСбыт

## Категории применения

### Для станков токарно-винторезных:
- Зубчатые передачи
- Подшипники
- Инструментальная оснастка

### Для револьверных головок:
- Диски
- Прижимы
- Манжеты
- Микропереключатели
- Резцовые блоки

### Для токарно-накатных станков (РЖД):
- Ролики упрочняющие
- Специализированная оснастка
