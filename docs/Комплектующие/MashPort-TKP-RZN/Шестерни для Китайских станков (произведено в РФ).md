# Шестерни для Китайских станков (произведено в РФ)

## Назначение
Шестерни для замены изношенных передач на китайских токарных станках. Полная совместимость с оборудованием иностранного производства. Производство в РФ по оригинальным чертежам и техническим условиям.

## Технические характеристики
- Диаметр: 20–200 мм  
- Модуль: 1,0–5,0 мм  
- Материал: Сталь конструкционная (ГОСТ 4543-71), чугун (ГОСТ 1215-79)  
- Класс точности: 7–8 (ГОСТ 1643-81)  
- Твердость: HB 160–220  
- Ресурс: 10 000–20 000 часов  

## Типы шестерен
- Прямозубые цилиндрические  
- Косозубые цилиндрические  
- Конические с прямым зубом  
- Конические с круговым зубом  
- Червячные пары  
- Специальные исполнения под заказ
