# Генерация ТКП (Технико-коммерческое предложение)

## Описание

Скрипт `generate_tkp.py` автоматически генерирует файлы ТКП (Технико-коммерческое предложение) из эталонного каталога товаров. 

ТКП представляет собой пронумерованный список всех офферов в формате:
```
Номер) SKU — Название товара (Категория)
```

## Требования

- Python 3.6+
- pandas

## Установка зависимостей

```bash
pip install pandas
```

## Использование

```bash
python generate_tkp.py
```

## Выходные файлы

Скрипт создает два файла:

1. **ТКП всех офферов.txt** - полный список всех товаров из каталога (494 оффера)
2. **ТКП станков.txt** - список только товаров, связанных со станками (484 оффера)

## Формат выходных файлов

Каждая строка имеет формат:
```
Номер) SKU — Название (Категория)
```

Пример:
```
1) BARABAN.FARTUK.16K20 — Барабан - копир на фартук станка 16К20 (Фартуки)
2) BLOKZUB.FARTUK.1M63.16360281 — Блок зубчатый 1М63Б_60_281 (Фартук, модель 1М63) (Кулачки)
```

## Исходные данные

Скрипт использует файл `catalogs/ЭТАЛОННЫЙ есть UID и SKU22.12.csv` как источник данных.

## Фильтрация для станков

Для файла "ТКП станков" применяется фильтрация по ключевым словам:
- станок
- станки
- токарн
- фрезерн
- сверлильн

Товары, содержащие эти слова в названии или категории, включаются в файл.

## Настройка

Для изменения источника данных или выходных файлов, отредактируйте константы в начале файла `generate_tkp.py`:

```python
CATALOG_FILE = BASE_DIR / "catalogs" / "ЭТАЛОННЫЙ есть UID и SKU22.12.csv"
OUTPUT_FILE_ALL = BASE_DIR / "ТКП всех офферов.txt"
OUTPUT_FILE_MACHINES = BASE_DIR / "ТКП станков.txt"
```
