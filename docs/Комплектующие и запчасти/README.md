# Комплектующие и запчасти для токарных станков

Данный раздел содержит офферы (торговые предложения) на комплектующие и запчасти для токарных станков, включая детали с сайта tdrusstankosbyt.ru и других источников ТКП.

## Категории комплектующих и запчастей

### Винты
- Винты пиноли
- Трапецеидальные винты суппортные

### Рейки
- Рейки зубчатые различных типоразмеров

### Каретка и суппорт
- Каретки суппорта для различных моделей станков
- Суппорты в сборе

### Муфты обгонные
- Муфты обгонные для станков 1М63, 1М65 и их модификаций
- Муфты обгонные для станков 16Р25П, 16Р25

### Гайки маточные
- Маточные гайки для различных моделей токарных станков
- Чугунные корпуса маточных гаек

### Фрикцион в сборе
- Фрикционные муфты
- Диски фрикционные

### Колеса
- Колеса конические (конические шестерни)
- Колеса зубчатые для шпиндельных бабок и кареток

### Червячные пары
- Червячные пары для токарных станков

### Гайки
- Гайки для узла задняя бабка
- Гайки различного назначения

### Комплектующие к узлам
- Общие комплектующие для станков токарной группы

### Патроны
- Патроны 3-х и 4-х кулачковые самоцентрирующиеся
- Комплекты кулачков
- Комплекты винтов

### Кулачки
- Кулачки для патронов различных станков
- Кулачки с винтами

### Захваты и вкладыши
- Захваты инструмента
- Изготовление захватов и вкладышей по специальным чертежам

### Венцы
- Венцы зубчатые для токарных станков

### Ползушки
- Отливки кареток, ползушек, суппортов

### Сухари
- Сухари регулировочные
- Клинья регулировочные

## Источники

Данные офферы собраны из:
- ТКП (технико-коммерческие предложения) станков
- tdrusstankosbyt.ru
- Каталоги запчастей для станков
