# Ролики сглаживающие

Технико-коммерческие предложения на ролики сглаживающие для токарно-накатных станков обработки железнодорожных осей.

## 📋 Каталог ТКП

### Ролики сглаживающие для станка РТ301.01

**Файл:** [Ролики сглаживающие для станка РТ301.01.md](./Ролики%20сглаживающие%20для%20станка%20РТ301.01.md)

**Назначение:** Финишная обработка подступичной части железнодорожных осей методом поверхностного пластического деформирования

**Основные характеристики:**
- Диаметр ролика: Ф110 мм
- Материал: Сталь ШХ15, 9ХС
- Твёрдость: HRC 56-60
- Достигаемая шероховатость: Ra 0.63-1.25 мкм
- Ресурс: до 8000 часов

**Применение:**
- Сглаживание микронеровностей после накатки
- Формирование требуемой шероховатости
- Стабилизация остаточных напряжений
- Подготовка к контрольным операциям

## 🎯 О роликах сглаживающих

Ролики сглаживающие — критически важный компонент технологического процесса обработки железнодорожных осей. Они обеспечивают:

✅ **Высокое качество поверхности** — Ra 0.63-1.25 мкм  
✅ **Равномерность упрочнения** — стабильные остаточные напряжения  
✅ **Долговечность осей** — улучшенные эксплуатационные характеристики  
✅ **Соответствие стандартам РЖД** — полное соответствие требованиям

## 🔧 Комплектация

Каждый ролик поставляется в сборе:
- Рабочий ролик (закалённая сталь)
- Подшипниковые узлы (5 класс точности)
- Крепёжные элементы
- Защитные уплотнения
- Документация и сертификаты
