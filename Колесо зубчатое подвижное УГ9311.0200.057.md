# Колесо зубчатое подвижное УГ9311.0200.057

## Описание

Колесо зубчатое подвижное УГ9311.0200.057 - специализированная деталь коробки подач токарного станка серии УГ9311. Представляет собой зубчатое колесо, установленное на шлицевом валу с возможностью осевого перемещения для включения различных передач. Обеспечивает изменение передаточного отношения и скорости вращения шпинделя.

## Технические характеристики

- **Применение**: Токарные станки серии УГ9311
- **Материал**: Легированная сталь 40Х, 40ХН с цементацией и закалкой зубьев
- **Точность**: Степень точности по ГОСТ 1643-81 - 7-8 степень
- **Ресурс работы**: Не менее 10 000 часов работы
- **Особенности**: Внутренние шлицы для установки на вал, высокая износостойкость зубьев

## Назначение

Колесо зубчатое подвижное предназначено для:
- Переключения передач в коробке подач
- Изменения скорости вращения шпинделя
- Изменения величины подачи
- Реализации различных режимов обработки
- Осевого перемещения для ввода в зацепление
