# Инструкция по загрузке тестовой страницы в Tilda

1. Выберите один из сгенерированных JSON-файлов в папке `catalogs/pages/` (например, `stanokrtrt1m63.json`).
2. Войдите в Tilda и создайте новую страницу на основе шаблона.
3. Заполните блоки страницы данными из выбранного JSON-файла:
   - Заголовок, подзаголовок, описание
   - Таблицу характеристик
   - Комплектацию, преимущества, фото
   - Связанные товары (related_products)
4. Сохраните и опубликуйте страницу.
5. Проверьте корректность отображения всех блоков и ссылок.
