# Инструкция по массовой загрузке 100 страниц в Tilda

1. Откройте Tilda Store или раздел импорта каталога.
2. Выберите файл `TILDA_IMPORT_100_PAGES.csv` из папки `catalogs/`.
3. Следуйте инструкциям Tilda для импорта товаров/страниц из CSV.
4. После загрузки проверьте, что все страницы созданы и заполнены корректно.
5. При необходимости вручную скорректируйте отдельные страницы или блоки.
6. Для перелинковки используйте поле `related_products` и таблицу соответствия URL.
