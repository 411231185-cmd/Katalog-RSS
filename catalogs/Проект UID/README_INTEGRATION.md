# 📦 ИНТЕГРАЦИЯ UID/SKU В КАТАЛОГ СТАНКОАРТЕЛЬ

## 📋 Что это?

Полный набор инструментов для проверки, исправления и вставки **UID/SKU кодов** всех 336+ артикулов из каталога РУССтанкоСБыт в вашу базу данных.

---

## 📂 СТРУКТУРА ФАЙЛОВ

```
├── UIDSKU_ABSOLUT_FINAL.csv      ← Исходный CSV файл с UID/SKU (336 записей)
├── UIDSKU_VERIFIED.csv            ← Проверенный и очищенный CSV
├── integration_script.py           ← Python скрипт для автоматической интеграции
├── run_integration.bat             ← Батник для запуска скрипта (Windows)
├── integration_database.sql        ← SQL скрипты для разных БД
└── README.md                       ← Этот файл
```

## 🚀 БЫСТРЫЙ СТАРТ

### Вариант 1: Автоматический (Рекомендуется)

1. **Скачай все файлы** в одну папку:
   ```
   C:\Users\User\Dropbox\ООО РУССтанкоСБыт\ИНТЕГРАЦИЯ\
   ```

2. **Отредактируй** `integration_script.py`:
   - Строка 11: Укажи свой путь к БД вместо `[YOUR_DATABASE_PATH]`

3. **Запусти** `run_integration.bat` двойным кликом

4. **Готово!** Скрипт создаст резервную копию и вставит все данные

### Вариант 2: Ручной (SQL)

1. **Открой свою БД** (MySQL, PostgreSQL, SQLite и т.д.)

2. **Скопируй нужный SQL-скрипт** из `integration_database.sql`

3. **Выполни запрос** в консоли БД

4. **Проверь результат**:
   ```sql
   SELECT COUNT(*) FROM catalog_items;
   ```

### Вариант 3: Через Excel/LibreOffice

1. **Открой** `UIDSKU_VERIFIED.csv` в Excel
2. **Скопируй колонки** UID и SKU
3. **Вставь** в свою систему каталогизации

## 🔧 КАК РАБОТАЕТ Python СКРИПТ?

```python
1. Читает UIDSKU_ABSOLUT_FINAL.csv
2. Проверяет каждую строку на ошибки
3. Создаёт резервную копию существующей БД
4. Объединяет новые данные с существующими
5. Записывает результат в БД
6. Логирует все действия в integration_log.txt
```

## 📝 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Пример 1: Поиск по UID
```sql
SELECT * FROM catalog_items 
WHERE uid LIKE '%STANOK%';
```

### Пример 2: Поиск артикула 16K20
```sql
SELECT uid FROM catalog_items 
WHERE sku = '16K20';
-- Результат: STANOK.16K20
```

### Пример 3: Все зубчатые колёса
```sql
SELECT uid, sku FROM catalog_items 
WHERE category = 'KOLZUB'
ORDER BY sku;
```

### Пример 4: Статистика по категориям
```sql
SELECT category, COUNT(*) as count 
FROM catalog_items 
GROUP BY category 
ORDER BY count DESC;
```


```
CSV файл должен быть в кодировке UTF-8
Открой в Notepad++ → Encoding → UTF-8
```
