# Задние бабки

## Описание

Задние бабки для токарных станков различных моделей. Узел предназначен для поддержки длинных заготовок при обработке и размещения инструмента (сверл, зенкеров, разверток) для обработки центровых отверстий. Включает комплект сменных шестерен для различных моделей станков.

## Технические характеристики

- **Применение**: Токарные станки моделей 165, 1658, 1А64, 1Н983, РТ502, РТ5001, РТ5004
- **Материал**: Чугун СЧ20, стальные детали - сталь 45, 40Х
- **Точность**: Соосность центров не более 0.01 мм на длине 1000 мм
- **Ресурс работы**: Не менее 10 лет при правильной эксплуатации
- **Особенности**: Регулировка положения по высоте, фиксация пиноли

## Назначение

Задняя бабка обеспечивает:
- Поддержку свободного конца длинной заготовки при точении
- Установку режущего инструмента для обработки отверстий
- Точное центрирование детали
- Возможность смещения корпуса для обточки конусов

## Преимущества

- Высокое качество изготовления
- Соответствие ГОСТ и ТУ
- Гарантия производителя
- Наличие сменных шестерен для различных моделей
- Взаимозаменяемость деталей

## Область применения

**Модели станков:**
- 165 - токарно-винторезный станок
- 1658 - токарно-винторезный станок повышенной точности
- 1А64 - токарно-винторезный станок
- 1Н983 - токарный многорезцовый полуавтомат
- РТ502, РТ5001, РТ5004 - токарные станки

**Сменные шестерни для задних бабок:**
- 165_08_177, 165_08_178 (модель 165)
- 1658_15_175, 1658_15_184, 1658_15_186, 1658_15_187, 1658_15_195 (модель 1658)
- 1А64_03_210, 1А64_03_211, 1А64_03_212, 1А64_03_264, 1А64_03_274 (модель 1А64)
- 1Н983_34_150, 1Н983_34_154 (модель 1Н983)
