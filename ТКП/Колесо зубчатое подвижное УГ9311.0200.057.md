# Колесо зубчатое подвижное УГ9311.0200.057

## Описание

Колесо зубчатое подвижное УГ9311.0200.057 для токарно-револьверного станка с ЧПУ модели УГ9311. Подвижное зубчатое колесо используется в механизмах переключения скоростей или подач, перемещается вдоль вала для зацепления с различными шестернями. Изготавливается по оригинальным чертежам с высокой точностью.

## Технические характеристики

- **Применение**: Для станка УГ9311
- **Материал**: Легированная сталь 40Х, 40ХН
- **Точность**: 6-7 степень точности по ГОСТ
- **Термообработка**: Цементация или закалка ТВЧ
- **Твердость**: HRC 56-62 для зубчатого венца
- **Особенности**: Шлицевое или шпоночное соединение с валом

## Назначение

Используется для переключения передач в коробке скоростей или подач станка.
