# Муфты обгонные

## Описание

Производство и поставка обгонных муфт для металлорежущих станков. Обгонные муфты (муфты свободного хода) позволяют передавать крутящий момент только в одном направлении, автоматически отключаясь при обратном вращении.

## Технические характеристики

- **Применение**: Для различных станков и механизмов
- **Тип**: Роликовые, храповые
- **Материал**: Легированная сталь с термообработкой
- **Момент**: Согласно типоразмеру
- **Ресурс**: Не менее 10000 моточасов

## Назначение

Передача крутящего момента в одном направлении, защита механизмов.

## Преимущества

- Автоматическое срабатывание
- Надежность
- Долговечность
- Компактность

## Область применения

Станки, приводы, подъемные механизмы.

## Комплектация

- Обгонная муфта
- Крепежные элементы
