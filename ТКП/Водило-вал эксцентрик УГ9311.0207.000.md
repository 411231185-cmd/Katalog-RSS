# Водило-вал эксцентрик УГ9311.0207.000

## Описание

Водило-вал эксцентрик УГ9311.0207.000 для токарно-револьверного станка с ЧПУ модели УГ9311. Деталь представляет собой сложную конструкцию, объединяющую функции водила планетарной передачи, вала и эксцентрика. Изготавливается по оригинальным чертежам с высокой точностью обработки всех функциональных поверхностей.

## Технические характеристики

- **Применение**: Для токарно-револьверного станка УГ9311
- **Материал**: Легированная сталь 40Х, 40ХН
- **Точность**: 6 степень точности
- **Термообработка**: Закалка и отпуск, локальная закалка ТВЧ
- **Твердость**: HRC 45-52 для основных поверхностей
- **Особенности**: Сложная геометрия, требующая прецизионной обработки

## Назначение

Водило-вал эксцентрик используется для:
- Передачи вращательного движения в планетарной передаче
- Создания эксцентричного движения
- Обеспечения работы специфических механизмов станка
- Точного позиционирования рабочих органов
