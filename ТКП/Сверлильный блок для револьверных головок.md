# Сверлильный блок для револьверных головок

## Описание

Сверлильные блоки для револьверных головок токарных станков с ЧПУ. Предназначены для установки сверл, зенкеров, разверток и другого осевого инструмента в позиции револьверной головки. Обеспечивают точное базирование и надежное крепление инструмента с коническими хвостовиками.

## Технические характеристики

- **Применение**: Для револьверных головок 16К30Ф3, 16М30Ф3, 1П756ДФ3, РТ755Ф
- **Конус**: Морзе 1, 2, 3 или метрический
- **Точность базирования**: Биение не более 0.02 мм
- **Материал**: Легированная сталь с термообработкой
- **Особенности**: Возможность использования инструмента с различными конусами

## Назначение

Сверлильный блок используется для:
- Сверления отверстий
- Зенкерования
- Развертывания
- Нарезания внутренней резьбы
- Растачивания отверстий

## Преимущества

- Высокая точность базирования инструмента
- Универсальность применения
- Надежная фиксация конического хвостовика
- Быстрая смена инструмента
- Совместимость с различными револьверными головками

## Дополнительная информация

Сверлильные блоки изготавливаются с высокой точностью обработки конического отверстия для обеспечения минимального биения инструмента. Предлагаем блоки под различные конусы Морзе и метрические конусы.

Возможно изготовление специальных сверлильных блоков под конкретные требования заказчика, в том числе блоков с вращающимся инструментом (с приводом).

При подборе необходимо указать модель револьверной головки и тип конуса используемого инструмента.
