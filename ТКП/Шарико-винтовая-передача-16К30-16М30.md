# Шарико-винтовая передача 16К30, 16М30

## Описание

Производство и поставка шарико-винтовых пар (ШВП) специально адаптированных для токарных станков с ЧПУ моделей **16К30** и **16М30**. ШВП преобразует вращательное движение в поступательное с минимальными потерями на трение и обеспечивает высокую точность позиционирования инструмента в револьверной головке.

## Технические характеристики

| Параметр                  | Значение                                |
| ------------------------- | --------------------------------------- |
| **Модели применения**     | 16К30, 16К30Ф3, 16М30, 16М30Ф3, 16А20Ф3 |
| **Диаметр винта**         | 16–32 мм                                |
| **Шаг резьбы**            | 2–5 мм                                  |
| **Точность**              | Класс C5–C7 (по ГОСТ)                   |
| **Люфт**                  | ≤ 0.1 мм                                |
| **Предварительный натяг** | Есть / Регулируемый                     |
| **Преднагрузка**          | 5–15 кН                                 |
| **Ресурс работы**         | 20000–50000 км пробега                  |

## Назначение

Шарико-винтовые пары для станков 16К30 и 16М30 предназначены в первую очередь для:

- **Расположение оси инструмента** в револьверной головке (осевое позиционирование)
- **Быстрая и точная смена позиции** инструмента (время позиционирования ≤ 1.2 сек)
- **Микроперемещение резцового блока** для компенсации износа
- **Прецизионное интерполирование** по координатам X, Y, Z
- **Обеспечение высокой точности обработки** детей (±0.001 мм)
- **Снижение вибраций** и шума при работе

## Преимущества

✅ **Высокий КПД** — 90–95%, минимальные потери на трение

✅ **Точность позиционирования** — ±0.001 мм обеспечивается без люфта

✅ **Надежность** — гарантирует долгий срок службы без обслуживания до 20000 км пробега

✅ **Плавность хода** — исключает рывки и самопроизвольные смещения

✅ **Универсальность** — совместимо с 16К30, 16М30 и модификациями (Ф3, Ф2)

✅ **Быстрая смена инструмента** — смена позиции за 0.8–1.2 сек

## Применение

### Модели станков

- токарные станки **16К30** (универсальные, с ЧПУ)
- токарные станки **16М30** (универсальные, с ЧПУ)
- токарные станки **16К30Ф3** (с ЧПУ — Фанук)
- токарные станки **16М30Ф3** (с ЧПУ — Фанук)
- обрабатывающие центры **16А20Ф3**

### Узлы применения

- **Революрационная головка** — главный механизм для смены инструмента
- **Шпиндельная бабка** — для осевого позиционирования
- **Суппорт** — продольное перемещение
- **Координатный стол** — прецизионные перемещения

## Комплектация

### Стандартная поставка

- ✓ Винт шарико-винтовой (обработанный, отполированный)
- ✓ Гайка шариковая с шариками и трубками рециркуляции
- ✓ Подшипники упорно-радиальные концевые
- ✓ Уплотнители (сальники) защитные
- ✓ Гофры и защитные колпаки

### Комплект по заказу

- опорные подушки (фиксированная / плавающая)
- муфты соединительные упругие
- фланцы монтажные под станок
- смазка специальная для ШВП (консистентная / жидкая)
- узлы концевых датчиков (в сборе)

## Область применения

**Типы обработки:**

- токарная обработка деталей из стали, чугуна, цветных металлов
- фрезерование и сверление на станках с ЧПУ
- шлифование и притирка
- суперфиниширование и полирование

**Примеры деталей:**

- валы и оси
- кольца и втулки
- шестерни и венцы
- диски и маховики
- корпуса и кожухи
