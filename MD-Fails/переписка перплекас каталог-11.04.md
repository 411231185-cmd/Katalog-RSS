```
Directus-поле       →   Promportal-колонка

```

# теперь нужно сделать это -Подключение кода ретаргетинга VK

В рекламном кабинете Вконтакте добавьте новый пиксель. Для этого перейдите на страницу «Сайты» и нажмите на кнопку «Добавить пиксель».

Заполните поле «Домен сайта».

Нажмите на кнопку «Получить код».

Из представленного кода необходимо скопировать набор символов после «_tmr.push({id:»

В личном кабинете на PromPortal.su скопированное значение вставьте в форму ниже и нажмите «Сохранить».

Это не техническая настройка — это пошаговая инструкция для ручных действий в двух личных кабинетах. Вот чёткий план:

***

## Подключение пикселя ретаргетинга VK к Promportal.su

### Шаг 1 — Создайте пиксель в рекламном кабинете VK

1. Откройте **[vk.com/ads](https://vk.com/ads)** → войдите в рекламный кабинет
2. В левом меню нажмите **«Сайты»**
3. Нажмите **«Добавить пиксель»**
4. Заполните поле **«Домен сайта»** — введите домен вашей страницы на Promportal:

```
tdrusstankosbyt.promportal.su
```

*(или ваш кастомный домен если уже привязали)*
5. Нажмите **«Получить код»**

***

### Шаг 2 — Скопируйте ID пикселя

В появившемся коде найдите строку:

```javascript
_tmr.push({id: XXXXXXXX, type: "pageView"...
```

Скопируйте только **цифры** — это и есть ваш ID пикселя.

> На скриншоте из примера ID выглядит так: `3417028` — у вас будет своё число.

***

### Шаг 3 — Вставьте ID в Promportal

1. Войдите в личный кабинет **promportal.su**
2. Перейдите: **Настройки → SEO-настройки** (или **Реклама → Ретаргетинг**)
3. Найдите поле **«Код ретаргетинга VK»** или **«VK Pixel ID»**
4. Вставьте скопированный ID (только цифры, без кавычек)
5. Нажмите **«Сохранить»**

***

### ✅ Проверка работы пикселя

После установки проверьте через расширение **[VK Pixel Helper](https://chrome.google.com/webstore/detail/vk-pixel-helper/)** для Chrome:

- Откройте вашу страницу на Promportal
- Расширение покажет зелёный значок если пиксель работает корректно

***

### 💡 Зачем это нужно

| Возможность | Результат |
| :-- | :-- |
| Ретаргетинг на посетителей страницы | Показываем рекламу тем, кто уже смотрел ваши товары |
| Аудитории по категориям товаров | Отдельные кампании для тех, кто смотрел суппорты / шестерни / ЧПУ |
| Look-alike аудитории | VK найдёт похожих на ваших покупателей |
| Отслеживание заявок | Видим откуда пришли лиды |

# и это тоже

Вижу — это форма **«Новый каталог»** в VK Рекламе. Это подключение товарного фида к рекламным кампаниям VK. Вот полная инструкция:

***

## Подключение каталога товаров к VK Рекламе

### Что выбрать в форме

На скриншоте два варианта — **выбирайте «Вручную»**, не «Фид или сообщество».

> Почему: у вас товары на Promportal, а не в сообществе ВКонтакте. Нужно загрузить XML-фид с Promportal.

***

### Шаг 1 — Получите ссылку на фид с Promportal

1. Войдите в личный кабинет **promportal.su**
2. Перейдите: **Настройки → Экспорт / Фиды**
3. Найдите раздел **«Фид для ВКонтакте»** или **«YML / XML фид»**
4. Скопируйте ссылку вида:

```
https://tdrusstankosbyt.promportal.su/export/vk/feed.xml
```


***

### Шаг 2 — Заполните форму в VK Рекламе

| Поле | Что вводить |
| :-- | :-- |
| **Название** | `ТД РУССтанкоСбыт — Запчасти токарные 2026` |
| **Добавление позиций** | Выбрать **«Вручную»** |
| **Ссылка на фид** | Вставить ссылку с Promportal |
| **Период обновления** | **Ежедневно** (оставить как есть) |
| **Автоматически удалять UTM-метки** | ☑️ Включить — чтобы UTM от Promportal не конфликтовали |


***

### Шаг 3 — Нажмите «Создать каталог»

После создания VK начнёт загрузку товаров из фида. Обычно занимает **10–30 минут**.

***

### Что делать если фида нет на Promportal

Если Promportal не даёт готовую ссылку на VK-фид — сообщите, и я сгенерирую **XML-файл** в нужном формате VK из вашего CSV-каталога, который вы разместите на любом хостинге.
