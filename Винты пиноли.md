# Винты пиноли

## Описание

Винты пиноли для задних бабок токарных станков с трапецеидальной резьбой. Обеспечивают точное перемещение пиноли задней бабки при поддержке заготовки или установке инструмента для обработки отверстий.

## Технические характеристики

- **Применение**: Задние бабки токарных станков различных моделей
- **Материал**: Сталь 45 с термообработкой
- **Резьба**: Трапецеидальная TR18×4 и другие типоразмеры
- **Точность**: Класс точности резьбы 7H по ГОСТ 9562
- **Ресурс работы**: Не менее 10 000 циклов перемещения
- **Особенности**: Износостойкая резьба, точная геометрия

## Назначение

Винт пиноли предназначен для:
- Осевого перемещения пиноли задней бабки
- Точного подвода центра к заготовке
- Фиксации пиноли в рабочем положении
- Выдвижения пиноли для установки инструмента
- Обеспечения плавного хода механизма

## Преимущества

- Высокое качество изготовления
- Соответствие ГОСТ и ТУ
- Гарантия производителя
- Термическая обработка для износостойкости
- Точность резьбы
- Совместимость с различными моделями станков

## Область применения

**Модели станков:**
- Токарные станки различных моделей
- Универсальные токарно-винторезные станки
- Станки повышенной точности

**Стандартные типоразмеры:**
- TR18×4 - наиболее распространенный размер
- TR20×4, TR22×5 - для станков большей размерности
- Другие типоразмеры по требованию
