# Револьверные головки

## Описание

Револьверные головки для токарных станков - многопозиционное устройство для крепления и последовательного использования нескольких режущих инструментов без остановки станка. Обеспечивают быструю смену инструмента и высокую производительность обработки.

## Технические характеристики

- **Применение**: Токарные станки с ЧПУ и универсальные токарные станки
- **Материал**: Корпус - высокопрочный чугун, механизмы - закаленная сталь
- **Точность**: Биение торца головки не более 0.01 мм
- **Ресурс работы**: Не менее 100 000 циклов индексации
- **Особенности**: Быстрая индексация, надежная фиксация позиций

## Назначение

Револьверная головка предназначена для:
- Установки нескольких режущих инструментов одновременно
- Автоматической смены инструмента в процессе обработки
- Повышения производительности токарной обработки
- Сокращения вспомогательного времени на смену инструмента
- Выполнения комплексной обработки детали за один установ

## Преимущества

- Высокое качество изготовления
- Соответствие ГОСТ и ТУ
- Гарантия производителя
- Точная индексация позиций
- Надежная фиксация инструмента
- Возможность установки различных типов инструмента

## Область применения

**Модели револьверных головок:**
- 16К30Ф3.40.000 - для станков 16К30Ф3
- 16М30Ф3.40.000 - для станков 16М30Ф3
- 1П756ДФ3.40.000 - для станков 1П756ДФ3
- 1П756ДФ3.39.000 - для станков 1П756ДФ3

**Типы станков:**
- Токарные станки с ЧПУ моделей 16К30Ф3, 16М30Ф3
- Токарные автоматы и полуавтоматы 1П756ДФ3
- Многошпиндельные токарные автоматы
