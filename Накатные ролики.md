# Накатные ролики

## Описание

Накатные ролики для токарных станков, в том числе для модели UBB-112. Предназначены для нанесения рифлений (накаток) на поверхность деталей для улучшения сцепления при захвате руками или для декоративных целей.

## Технические характеристики

- **Применение**: Токарные станки, накатные устройства
- **Материал**: Быстрорежущая сталь Р6М5, Р18 (закаленная)
- **Твердость**: HRC 58-62 после термообработки
- **Точность**: Профиль зубьев по ГОСТ 9484
- **Ресурс работы**: До 50 000 деталей (в зависимости от материала)
- **Особенности**: Различные профили накатки (прямая, перекрестная, ромбическая)

## Назначение

Накатные ролики предназначены для:
- Нанесения рифлений на цилиндрические поверхности
- Создания нескользящей поверхности на рукоятках
- Декоративной отделки деталей
- Увеличения диаметра детали методом пластической деформации
- Упрочнения поверхностного слоя

## Преимущества

- Высокое качество изготовления
- Соответствие ГОСТ и ТУ
- Гарантия производителя
- Закаленная рабочая поверхность
- Точность профиля
- Долгий срок службы

## Область применения

**Типы накаток:**
- Прямая накатка (продольные рифления)
- Перекрестная накатка (ромбический рисунок)
- Сетчатая накатка
- Специальные профили

**Модели станков:**
- UBB-112 (ролик накатной закаленный)
- Токарные станки всех моделей
- Специальные накатные станки

**Области применения:**
- Изготовление рукояток инструмента
- Производство крепежных изделий
- Декоративная обработка
- Изготовление рифленых валов

## Комплектация

- Накатной ролик (закаленный)
- Упаковка для транспортировки

Ролики могут поставляться парами для перекрестной накатки.
