# Spoki Noki Logger

VS Code-расширение для автоматического сохранения лога чата/документа в Markdown-файл при появлении триггера "споки ноки".

## Возможности
- Мониторит все изменения текста в редакторе.
- При появлении триггера (по умолчанию "споки ноки") сохраняет весь текст текущего документа в MD-файл с уникальным именем (chat-logs/MD-Spoki-Noki-YYYY-MM-DD-HHMM.md).
- Создаёт папку, если её нет.
- Показывает уведомление о сохранении.
- Гибко настраивается через settings.json (папка, триггер).

## Установка и запуск
1. В папке расширения:
   - `npm install`
   - `npm run compile`
2. F5 в VS Code (Debug Extension) — откроется новое окно VS Code с расширением.
3. Настройки: Ctrl+Shift+P → Preferences: Open Settings (JSON), добавить:
```json
"spokiNoki.folder": "chat-logs",
"spokiNoki.trigger": "споки ноки"
```
4. Откройте Markdown или текстовый файл, напишите "споки ноки" — лог сохранится автоматически.

## Пример
- Вставьте триггер в любой документ:
  > споки ноки
- В папке chat-logs появится файл MD-Spoki-Noki-YYYY-MM-DD-HHMM.md с содержимым документа.
