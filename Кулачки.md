# Кулачки

## Описание

Кулачки для токарных патронов - сменные зажимные элементы, непосредственно контактирующие с заготовкой. Изготавливаются в двух вариантах: основные (прямые) для наружного зажима и обратные для внутреннего зажима. Возможно изготовление специальных кулачков по индивидуальным чертежам для деталей сложной формы.

## Технические характеристики

- **Применение**: Токарные патроны всех типов и размеров
- **Материал**: Сталь 40Х, 45 с термообработкой (закалка рабочих поверхностей)
- **Точность**: Радиальное биение комплекта не более 0.03 мм
- **Ресурс работы**: Не менее 5 000 часов работы
- **Особенности**: Взаимозаменяемость, возможность переточки, специальные профили

## Назначение

Кулачки для патронов предназначены для:
- Непосредственного зажима заготовок в токарных патронах
- Центрирования деталей при установке
- Передачи усилия зажима на заготовку
- Закрепления деталей различной формы
- Защиты поверхности заготовки от повреждений (с накладками)
- Обеспечения надежной фиксации при обработке

## Область применения

**Типы кулачков:**
- Основные (прямые) - для наружного зажима
- Обратные - для внутреннего зажима
- Специальные - по чертежам заказчика
- Мягкие накладки - для защиты поверхности детали
- Профильные - для деталей сложной формы

**Патроны:**
- 3-х кулачковые самоцентрирующие
- 4-х кулачковые самоцентрирующие
- 4-х кулачковые независимые
- Патроны специального назначения

## Комплектация
