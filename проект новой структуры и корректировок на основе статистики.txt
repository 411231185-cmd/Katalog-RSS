## 2. Ключевые проблемы и предлагаемые корректировки

### 2.1. Автотаргетинг

**Наблюдение:** значительная часть бюджета уходит на группы с автотаргетингом, при этом конверсий по ним не зафиксировано, а общая конверсия кампании снижается[attached_file:61][attached_file:62].

**Предложение:**
- Снизить ставки по автотаргетингу минимум на 50 %.  
- Оставить одну тестовую группу с ограниченным бюджетом и контролем, остальные запросы вынести в отдельную тестовую кампанию или отключить.  
- Перераспределить высвободившийся бюджет на товарно‑ориентированные группы (шестерни по чертежам, револьверные головки, ремонт).

### 2.2. Ключевые слова без кликов / слабые запросы

**Наблюдение:** по ряду фраз есть показы и 0 кликов либо крайне низкий CTR (часть общих запросов по револьверным головкам, шестерням, ремонту)[attached_file:62].

**Предложение:**
- Вынести такие запросы в отдельный «тестовый блок» с минимальной ставкой.  
- Нецелевые запросы по смыслу добавить в минус‑слова.  
- Еженедельно чистить поисковые запросы и обновлять список минус‑слов.

### 2.3. Аналитика и техническое состояние сайта

**Наблюдение из Яндекс Вебмастера:**  
- Счётчик Метрики не привязан к сайту.  
- Отсутствуют/некорректны meta Description на многих страницах.  
- Нет рабочего Sitemap, не указан регион сайта[attached_file:63].

**Предложение:**
- Привязать счётчик Метрики к сайту и завести цели: отправка формы, звонок, клик по контактам; включить оптимизацию по целям в стратегии кампании.  
- Заполнить Description на основных посадочных страницах под рекламу.  
- Добавить Sitemap и указать регион или «Нет региона» (если целевая аудитория — вся РФ).

## 4. Формат взаимодействия

Предлагаем рассматривать этот документ как совместный проект новой структуры и корректировок:

1. Подтвердить/скорректировать:
   - бюджеты по группам;  
   - набор ключевых фраз и минус‑слов;  
   - ставки по приоритетным запросам.  
2. С вашей стороны — реализовать изменения в интерфейсе Директа/eLama.  
3. С нашей стороны — регулярно мониторить и анализировать статистику.Ключи Api нам тоже нужны от рекламной компании,которая будет по РСЯ и от метрики.
