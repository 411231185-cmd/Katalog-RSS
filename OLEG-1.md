# Структура каталога для нового сайта

Этот документ описывает структуру данных для импорта в новый каталог сайта.

---

## Описание колонок

| Колонка | Описание | Требование | Пример |
|---------|----------|------------|--------|
| **A: SKU (Артикул)** | Уникальный код товара | Критически важно! Не должно быть двух строк с одинаковым SKU. Только латиница. | `STANOK.16K20` |
| **B: Название** | Понятное название товара | Без лишних слов, максимально понятно | `Токарно-винторезный станок 16К20` |
| **C: Категория** | Полный путь к категории | Через символ `>>>` | `Станки>>>Токарные>>>Универсальные` |
| **D: Описание** | Маркетинговый текст | Можно с абзацами, без сложного форматирования (размеров шрифтов, цветов) | `Надежный станок для серийного производства...` |
| **E, F, G...** | Характеристики | Каждая характеристика — в отдельном столбце. Только цифры! | `Вес: 1500` `Мощность: 11` |
| **H: Совместимость** | Для запчастей | Артикулы станков через точку с запятой | `STANOK.16K20; STANOK.1M63` |

## Категории товаров

### 1. Станки

#### 1.1. Станки с ЧПУ
- Категория: `Станки>>>Станки с ЧПУ>>>Токарные`
- Примеры SKU:
  - `STANOK.CNC.16M30F3` - Станок 16М30Ф3
  - `STANOK.CNC.16A20F3` - Станок 16А20Ф3
  - `STANOK.CNC.RT755F3` - Станок РТ755Ф3
  - `STANOK.CNC.RT305M` - Станок РТ305М
  - `STANOK.CNC.RT779F3` - Станок РТ779Ф3

#### 1.2. Станки токарно-винторезные
- Категория: `Станки>>>Токарно-винторезные`
- Примеры SKU:
  - `STANOK.16K20` - Станок 16К20
  - `STANOK.16K40` - Станок 16К40
  - `STANOK.16P25` - Станок 16Р25
  - `STANOK.1M63N` - Станок 1М63Н (ДИП-300)
  - `STANOK.1N65` - Станок 1Н65 (ДИП-500)
  - `STANOK.RT117` - Станок РТ117
  - `STANOK.RT817` - Станок РТ817

#### 1.3. Станки для железнодорожной отрасли
- Категория: `Станки>>>Специальные>>>Железнодорожные`
- Примеры SKU:
  - `STANOK.RAIL.RT5001` - Лентобандажировочный РТ5001
  - `STANOK.RAIL.RT5003` - Лентобандажировочный РТ5003
  - `STANOK.RAIL.RT5004` - Бандажировочный РТ5004

## Правила формирования SKU

### Станки:
- Формат: `STANOK.[ТИП].[МОДЕЛЬ]`
- Примеры:
  - `STANOK.16K20` - токарный
  - `STANOK.CNC.16M30F3` - токарный с ЧПУ
  - `STANOK.RAIL.RT5001` - для ж/д отрасли
  - `STANOK.PIPE.1A983` - трубонарезной

### Револьверные головки:
- Формат: `TURRET.[МОДЕЛЬ]`
- Пример: `TURRET.16K30F3.40.000`

### Комплектующие:
- Формат: `[ТИП].[СПЕЦИФИКАЦИЯ]`
- Примеры:
  - `GEAR.Z45.M3` - шестерня
  - `BALLSCREW.40x10.L1500` - ШВП
  - `SHAFT.SPINDLE.1M63` - вал
  - `NUT.SPLIT.16K20` - гайка маточная

### Оснастка:
- Формат: `[ТИП].[РАЗМЕР/МОДЕЛЬ]`
- Примеры:
  - `CHUCK.3JAW.250` - патрон
  - `JAW.SOFT.SET3` - кулачки
  - `TOOLBLOCK.16K30F3.42.000` - резцовый блок

---

## Важные замечания

1. **SKU должны быть уникальными** - проверяйте перед добавлением нового товара
2. **Совместимость** - обязательно заполнять для всех запчастей и комплектующих
3. **Характеристики** - только числовые значения, единицы измерения в названии колонки
4. **Категории** - использовать строго установленную иерархию
5. **Описание** - максимально информативно, но без HTML и сложного форматирования
