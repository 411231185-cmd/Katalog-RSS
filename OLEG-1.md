# Полный каталог продукции для импорта

## Описание колонок

| Колонка | Описание и Требование | Пример |
|---------|----------------------|--------|
| **A: SKU (Артикул)** | Критически важно! Уникальный код товара. Не должно быть двух строк с одинаковым SKU. Используйте латиницу. | STANOK.16K20 |
| **B: Название** | Понятное название товара без лишних слов. | Токарно-винторезный станок 16К20 |
| **C: Категория** | Полный путь к категории через символ >>>. | Станки>>>Токарные>>>Универсальные |
| **D: Описание** | Маркетинговый текст. Можно с абзацами, но без сложного форматирования (размеров шрифтов, цветов). | Надежный станок для серийного производства... |
| **E, F, G... (Характеристики)** | Главное отличие: Каждая техническая характеристика — в отдельном столбце. Только цифры! | Вес: 1500, Мощность: 11 |
| **H: Совместимость** | (Для запчастей) Артикулы станков, к которым подходит деталь. Через точку с запятой. | STANOK.16K20; STANOK.1M63 |
