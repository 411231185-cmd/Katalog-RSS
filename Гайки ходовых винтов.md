# Гайки ходовых винтов

## Описание

Гайки для ходовых и винтовых передач металлорежущих станков. Трапецеидальные и метрические гайки для суппортов, задних бабок и других узлов. Изготавливаются из бронзы и чугуна.

## Технические характеристики

- **Применение**: Суппорты, задние бабки, столы станков
- **Материал**: Бронза БрАЖ9-4, БрОФ10-1, чугун СЧ20
- **Резьба**: Трапецеидальная Tr (ГОСТ 9484), метрическая
- **Точность**: Класс 7H по ГОСТ 9562
- **Ресурс работы**: До 15 000 моточасов
- **Особенности**: Возможность регулировки зазора (разрезные гайки)

## Назначение

Гайки ходовых винтов предназначены для:
- Преобразования вращения в поступательное движение
- Работы в паре с ходовыми винтами суппортов
- Перемещения задней бабки
- Обеспечения точности подачи
- Компенсации износа (разрезные конструкции)

## Область применения

**Типы гаек:**

**Бронзовые гайки:**
- Для ходовых винтов суппортов
- Высокая износостойкость
- Низкий коэффициент трения

**Чугунные гайки:**
- Для менее нагруженных узлов
- Экономичное решение

**Разрезные гайки:**
- С возможностью регулировки зазора
- Компенсация износа
- Устранение люфта

**Маточные гайки:**
- Для нарезания резьбы
- Установка на фартуке станка
- Включение/выключение

**Модели станков:**
- Токарные станки всех моделей
- Фрезерные станки
- Другие металлорежущие станки

**Узлы применения:**
- Продольная подача суппорта
- Поперечная подача
- Верхний суппорт
- Задняя бабка
- Столы фрезерных станков

## Комплектация

- Гайка с обработанной резьбой
- Регулировочные элементы (для разрезных гаек)
- Крепежные элементы
- Упаковка
