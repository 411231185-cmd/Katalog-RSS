# Шестерни для станков с ЧПУ

## Описание

Зубчатые колеса и шестерни для станков с ЧПУ различных моделей. Изготавливаются по оригинальным чертежам с соблюдением всех требований к точности зубчатых передач. Применяются в шпиндельных бабках, коробках подач, механизмах смены инструмента и других узлах станков с ЧПУ.

## Технические характеристики

- **Применение**: Станки с ЧПУ токарной и фрезерной группы
- **Материал**: Сталь 40Х, 45, 20ХН3А с термообработкой
- **Точность**: 6-8 степень точности по ГОСТ 1643-81
- **Ресурс работы**: До капитального ремонта станка
- **Особенности**: Высокая твердость зубьев (HRC 58-62), шлифованные зубья

## Назначение

Шестерни для станков с ЧПУ предназначены для:
- Передачи крутящего момента в механизмах станка
- Изменения частоты вращения исполнительных органов
- Обеспечения точного позиционирования
- Работы в системах главного привода
- Применения в коробках скоростей и подач

## Преимущества

- Высокое качество изготовления
- Соответствие ГОСТ и ТУ
- Гарантия производителя
- Термическая обработка зубьев
- Контроль геометрических параметров
- Низкий уровень шума при работе
- Высокая износостойкость

## Область применения

**Узлы станков:**
- Шпиндельные бабки
- Коробки подач
- Каретки суппортов
- Механизмы смены инструмента
- Приводы координатных осей

**Модели станков:**
- 164, 165 - токарные станки с ЧПУ
- 16К30, 16К40 - токарные станки с ЧПУ
- 1658 - токарный станок повышенной точности
- 1А64 - токарный станок
- Фрезерные станки с ЧПУ

## Комплектация

- Зубчатое колесо с механической обработкой
- Термическая обработка зубьев (при необходимости)
- Контрольный протокол
- Упаковка для транспортировки

Изготовление производится по чертежам заказчика или по каталожным номерам.
