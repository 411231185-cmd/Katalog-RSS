# Описания станков

Этот каталог содержит структурированные описания основных моделей станков.

## Структура

### Файлы

- **descriptions.json** - JSON формат для программной обработки
- **descriptions.csv** - CSV формат для импорта в Tilda/Excel
- **descriptions/*.md** - Отдельные Markdown файлы для каждого станка

## Формат данных

### JSON (descriptions.json)

```json
{
  "stanks": [
    {
      "sku": "1M63",
      "name": "Токарно-винторезный станок 1М63",
      "purpose": "Предназначен для выполнения...",
      "characteristics": [
        "Диаметр обработки: до 630 мм",
        "РМЦ: 1000-5000 мм",
        ...
      ],
      "equipment": ["Центры", "Патрон", ...],
      "source_url": "https://rosstanko.com/..."
    }
  ]
}
```

### CSV (descriptions.csv)

Колонки: SKU, Name, Purpose, Characteristics, Equipment, SourceURL

### Markdown (descriptions/*.md)

Каждый файл содержит:
- Заголовок с названием станка
- Назначение
- Основные характеристики (маркированный список)
- Комплектация (маркированный список)
- Источник информации

## Использование

### Обновление данных

Запустите скрипт для сбора данных с сайта:

```bash
python scrape_stanok_descriptions.py
```

### Ручное редактирование

Вы можете вручную отредактировать любой из Markdown файлов, затем регенерировать JSON и CSV:

```bash
python update_descriptions_from_markdown.py
```
