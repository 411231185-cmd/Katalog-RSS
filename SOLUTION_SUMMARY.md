# Решение задачи: ТКП всех офферов

## Задача
Создать ТКП (Технико-коммерческое предложение) на все офферы из каталога.

## Реализация

### 1. Создан скрипт `generate_tkp.py`
Автоматизированный инструмент для генерации ТКП из эталонного каталога.

**Возможности:**
- Чтение CSV каталога с автоопределением разделителя
- Генерация пронумерованных списков офферов
- Фильтрация по категориям (станки/все товары)
- Корректная обработка UTF-8 кодировки
- Логирование ошибок для отладки

### 2. Созданные файлы

#### ТКП всех офферов.txt
- **494 позиции** - полный список всех товаров из каталога
- Формат: `Номер) SKU — Название товара (Категория)`

#### ТКП станков.txt  
- **484 позиции** - только товары, связанные со станками
- Фильтрация по ключевым словам: станок, токарн, фрезерн, сверлильн

### 3. Формат выходных данных

```
1) BARABAN.FARTUK.16K20 — Барабан - копир на фартук станка 16К20 (Фартуки)
2) BLOKZUB.FARTUK.1M63.16360281 — Блок зубчатый 1М63Б_60_281 (Фартук, модель 1М63) (Кулачки)
...
494) SPINDELBABKA.VAL.1M63 — Вал шпиндельной бабки 1М63 (Шпиндельные бабки)
```

### 4. Документация

- **README.md** - обновлен с информацией о ТКП
- **docs/TKP_GENERATION.md** - подробная инструкция по использованию

## Использование

```bash
# Генерация ТКП файлов
python generate_tkp.py

# Результат:
# - ТКП всех офферов.txt (494 позиции)
# - ТКП станков.txt (484 позиции)
```
