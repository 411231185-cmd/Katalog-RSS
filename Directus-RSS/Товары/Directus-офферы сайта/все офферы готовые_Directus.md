## ПРАВИЛА ЗАПОЛНЕНИЯ ПОЛЕЙ

### 1. Category
— **ОСТАВЛЯЕШЬ ПУСТЫМ**
— Не заполняешь ничего

### 2. Compatible Products
— Ставь галочки у станков, указанных в разделе "Compatible Products" под каждым товаром
— Если станка нет в списке Directus — оставь поле пустым
— **НЕ ПРИДУМЫВАЙ** станки из головы
— Используй точные обозначения: 1М63, 1П756ДФ3, ДИП500 и т.д.

### 3. Slug
— Генерируй автоматически из Name на латинице
— Пример: `val-1n983-20-274` или `disk-instrumentalnyi-1p756df3-47-000`
— Используй транслитерацию через дефис

### 4. Images
— ПОКА ОСТАВЛЯЙ ПУСТЫМ (без картинок)

### 5. Description (ОСНОВНОЕ ОПИСАНИЕ)
— ВАЖНО: НЕ копируй весь текст из раздела "Description" целиком
— Форматируй по структуре ниже:

```

• Артикул: [указать]
• Материал: [указать]

• [Список станков и узлов]
```

### 6. Name
— Копируй точно как указано в поле Name из файла
— Без сокращений, без добавлений

### 7. Specs
— НЕ ЗАПОЛНЯЙ (оставь пустым)
— Это поле заполняется позже техническим специалистом

### 8. Price
— Оставь пустым (по запросу)

### 9. Is Published
— **ВКЛЮЧЕНО** (ставь галочку)

### 10. Image File (ГЛАВНОЕ ИЗОБРАЖЕНИЕ)
— **ОБЯЗАТЕЛЬНЫЙ ШАГ!**
— Нажми "Выбрать существующий"
— Перейди в библиотеку файлов Directus

— Альтернативный путь: https://admin.td-rss.ru/admin/files/7570d3bb-c49f-4152-bbb3-bc87886f737e

### 11. Meta Title
— Копируй точно из поля Meta Title
— Должен быть длиной 50-60 символов
— Формат: `[Название детали] для [станок] — в наличии, цена от производителя`

### 12. Meta Description
— Копируй точно из поля Meta Description  
— Должен быть длиной 150-160 символов
— Содержать гарантию **6 месяцев** (не 12!)
— Формат: `[Призыв к действию]. [Преимущества]. [Гарантия]. [Доставка]`
