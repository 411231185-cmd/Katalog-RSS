# Трапецеидальные винты суппортные

## Описание

Комплект трапецеидальных винтов для суппортов токарных станков. Обеспечивают продольное и поперечное перемещение резца с точной подачей. Изготавливаются с резьбой по ГОСТ 9484 для различных моделей станков.

## Технические характеристики

- **Применение**: Суппорты токарных станков всех моделей
- **Материал**: Сталь 45 с термообработкой
- **Резьба**: Трапецеидальная Tr по ГОСТ 9484
- **Класс точности**: 7H по ГОСТ 9562
- **Ресурс работы**: Не менее 15 000 моточасов
- **Особенности**: Закалка ходовой части резьбы, шлифовка

## Назначение

Трапецеидальные винты предназначены для:
- Продольного перемещения каретки суппорта
- Поперечного перемещения поперечных салазок
- Перемещения верхних салазок
- Точного позиционирования резца
- Обеспечения заданной подачи

## Преимущества

- Высокое качество изготовления
- Соответствие ГОСТ и ТУ
- Гарантия производителя
- Термообработка резьбы
- Точность шага резьбы
- Износостойкость
- Плавность хода

## Область применения

**Типы винтов в комплекте:**
- Винт продольной подачи (длинный)
- Винт поперечной подачи (средний)
- Винт верхнего суппорта (короткий)

**Модели станков:**
- 1М63, 16К20, 1К62
- 16К40, 16М30
- 1Н65, 1А64
- Универсальные токарные станки

**Стандартные резьбы:**
- Tr20×4, Tr22×5 - для продольной подачи
- Tr16×4, Tr18×4 - для поперечной подачи
- Tr14×3, Tr16×4 - для верхнего суппорта

## Комплектация

**Комплект винтов суппортных включает:**
- Винт продольной подачи с резьбой
- Винт поперечной подачи с резьбой
- Винт верхнего суппорта с резьбой
- Упаковка для транспортировки

Гайки к винтам поставляются отдельно или в комплекте по требованию.
