# Katalog RSS

Репозиторий для хранения и управления RSS каталогами из DropBox

## Структура репозитория

### 📁 catalogs/
Основные каталоги товаров в различных форматах

#### yml/
Каталоги в формате YML (Яндекс.Маркет)
- `merged.yml` - Объединённый каталог всех товаров
- `Prais-KONETs-DNIa-FINISh.yml` - Финальный прайс-лист
- `Shpindelnaia-babka-*.yml` - Каталоги шпиндельных бабок для различных моделей станков
- `Fartuk-1N65.yml` - Каталог фартуков

#### csv/
Экспортированные данные в формате CSV
- `merged_export.csv` - Базовый экспорт каталога
- `merged_qc_export.csv` - Расширенный экспорт с дополнительными полями (vendor, count, available)

#### excel/
- `Prais-Iandeks-Noiabr.xlsx` - Прайс для Яндекс.Маркет (ноябрь)

### 📁 matches/
Файлы сопоставления и соответствия товаров
- `matches-exact.txt` - Точные совпадения названий товаров
- `matches-partial.txt` - Частичные совпадения названий товаров

## Описание каталогов

Репозиторий содержит запасные части и оснастку для токарных станков:
- Шпиндельные бабки для моделей 1M63, 1A64, 1N983
- Фартуки для станков серии 1N65
- Патроны токарные (3-кулачковые, 4-кулачковые)
- Револьверные головки
- Суппорты, задние бабки
- Шестерни, винты, гайки
- Электрооборудование и комплектующие

## Форматы данных

### YML (Яндекс.Маркет)
Стандартный формат выгрузки для торговых площадок

### CSV
Табличный формат для импорта/экспорта:
- Базовый: id, name, price, currency, categoryId, url, vendorCode
- Расширенный: + vendor, count, available

### XLSX
Формат для работы в Excel с полным описанием товаров

## ТКП (Технико-коммерческое предложение)

### Файлы ТКП

В репозитории созданы следующие файлы ТКП:

1. **ТКП всех офферов.txt** - полный список всех товаров из каталога (494 позиции)
2. **ТКП станков.txt** - список товаров, связанных со станками (484 позиции)

### Формат

Каждая позиция представлена в формате:
```
Номер) SKU — Название товара (Категория)
```

Пример:
```
1) BARABAN.FARTUK.16K20 — Барабан - копир на фартук станка 16К20 (Фартуки)
```

### Генерация

Для регенерации файлов ТКП используйте скрипт:
```bash
python generate_tkp.py
```

Подробнее см. [docs/TKP_GENERATION.md](docs/TKP_GENERATION.md)
